Cumulative Distribution Functions
==============================================

Computing probabilities for continuous random variables requires integrating the PDF over intervals, 
which can become tedious when we need to calculate many different probabilities. Just as calculus 
provides us with antiderivatives to avoid repeated integration, probability theory offers us the 
**cumulative distribution function (CDF)** as a tool that eliminates the need for repeated 
integrations.

.. admonition:: Road Map ðŸ§­
   :class: important

   â€¢ Define the **cumulative distribution function (CDF)**.
   â€¢ Master the **relationship between PDF and CDF** through the fundamental theorem of calculus.
   â€¢ Learn the essential **properties of CDFs** that make them valid probability functions.
   â€¢ Practice **computing probabilities** using CDFs instead of direct integration.
   â€¢ Apply CDFs to find **percentiles and quantiles** for continuous distributions.

The Universal Language of Cumulative Probability
-------------------------------------------------

The **cumulative distribution function (CDF)** provides a unified approach to probability that 
works seamlessly for both discrete and continuous random variables. Instead of dealing 
with probability mass functions and probability density functions separately, the CDF 
gives us one consistent framework.

Definition
~~~~~~~~~~~~
For any random variable :math:`X` (discrete or continuous), the cumulative distribution 
function :math:`F_X(x)` is defined as:

.. math::
   F_X(x) = P(X \leq x)

The definition expresses a simple idea: :math:`F_X(x)` gives the probability 
that the random variable :math:`X` falls at or below any 
given value :math:`x`.

.. flat-table::
   :header-rows: 2
   :align: center
   :width: 100%

   * - :cspan:`1` Implementation of CDF to Different Variable Types
   
   * - Discrete 
     - Continuous

   * - .. math:: F_X(x) = P(X \leq x) = \sum_{t=-\infty}^{x} p_X(t)
     - .. math:: F_X(x) = P(X \leq x) = \int_{-\infty}^{x} f_X(t) \, dt

In both cases, the argument of the PMF/PDF is replaced with a dummy variable :math:`t`.
This is to avoid any confusion with the true argument of :math:`F_X`.

In the continuous case, :math:`F_X(x)` can be used for :math:`P(X\leq x)` or
:math:`P(X<x)`.

Requirements for a valid CDF
------------------------------

Cumulative distribution functions must satisfy several mathematical properties that 
reflect their probabilistic interpretation.

1. Monotonicity
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

As we move from a smaller to a larger :math:`x`, we can only **accumulate** the probability that :math:`X` falls 
below it. 
Therefore, :math:`F_X` is a non-decreasing function. That is, for any real numbers :math:`a < b`,

.. math::

   F_X(a) \leq F_X(b)

2. Limiting Behavior
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The accumulated probability must be close to 0 at a very small :math:`x` 
(no probability accumulates infinitely far to the left on the x-axis), 
and close to 1 for a large enough :math:`x`:

.. math::

   \lim_{x \to -\infty} F_X(x) = 0 \quad \text{and} \quad \lim_{x \to +\infty} F_X(x) = 1

3: Right Continuity
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

For any point :math:`c`,

.. math::

   \lim_{x \to c^+} F_X(x) = F_X(c).

This technical property ensures that the CDF behaves consistently at boundary points, 
particularly important when dealing with mixed discrete-continuous distributions or 
piecewise functions. This property is satisfied automatically for 
continuous distributions.

The PDF-CDF Connection
--------------------------------------------------

For continuous random variables, the relationship between a PDF and its CDF mirrors 
the fundamental theorem of calculus.

From PDF to CDF: Integration
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

By the definition,

.. math::

   F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt

:math:`F_X(x)` is the **accummulation** of all the probability from the left tail of 
the distribution up to the point :math:`x`.

From CDF to PDF: Differentiation
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Given a CDF :math:`F_X(x)`, we can recover the PDF by differentiation:

.. math::

   f_X(x) = \frac{d}{dx} F_X(x) = F_X'(x)

wherever the derivative exists. This relationship explains why the PDF measures 
the "rate of accumulation" of probability at each point.

Visualizing the Relationship
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The connection becomes clear when we graph the two functions together. 

.. _pdf-cdf-relationship:
.. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/pdf-cdf-comparison.png
   :alt: Relationship between PDF and CDF
   :align: center
   :figwidth: 70%

   Relationship between PDF and CDF

* If one function is piecewise-defined, the other is as well. They
  share the same set of boundaries.

* A probability of the form :math:`P(X\leq a)` corresponds to **the area to the left of**
  :math:`a` under a PDF, and to **the height of the curve** on a CDF.

* A PDF can take any shape, as long as it stays above the horizontal axis and integrates to 1.
  A CDF is more constrained in shape; it approaches 0 on the left, 1 on the right, and is
  never decreasing in between.

* In any region where the PDF is :math:`0`, the CDF is flat, but not necessarily 0.
  It retains any area accummulated in previous regions, but no area accummulates additionally. 


Building a Piecewise CDF
------------------------------

Constructing a CDF based on a piecewise-defined PDF requires careful attention.
The most crucial aspect of working with piecewise CDFs is remembering that they 
are **cumulative**â€”they must account for **all** probability accumulated up to each 
point, **not just in the current region**.

**Useful tips for constructing a piecewise CDF:**

* **Make a rough sketch** similar to :numref:`pdf-cdf-relationship`. Visually
  identify the regions where the CDF is approaching 0, incresing, flat, or
  approaching 1. At the end, check whether the computational result matches the sketch.

* **Stick to the definition** at first. By definition, 
   
  .. math:: F_X(x) = \int_{-\infty}^x f_X(t) \, dt

  is an integral from **negative infinity** up to the input value, :math:`x`.
  Starting directly from this definitionâ€”without skipping stepsâ€”will 
  help you become familiar with the subtle patterns involved in constructing piecewise CDFs.

.. admonition:: ExampleðŸ’¡: Constructing the CDF for a piecewise PDF
   :class: note 

   Construct the CDF of a random variable :math:`X` which has the
   following PDF. Use :numref:`pdf-cdf-relationship` as reference.

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{3x}{2} & \text{for } 0 \leq x \leq 1 \\
      0 & \text{for } 1 < x < 5 \\
      \frac{1}{4} & \text{for } 5 \leq x \leq 6 \\
      0 & \text{elsewhere}
      \end{cases}

   Work on each region given by the PDF separately:

   **Region 1** (:math:`x < 0`): No probability accumulated yet

      .. math:: 
         F_X(x) = \int_{-\infty}^x f_X(t) \, dt = \int_{-\infty}^x 0 \, dt = 0

   **Region 2** (:math:`0 \leq x \leq 1`): Accumulating triangular area
      
      .. math:: 
         F_X(x) = \int_{-\infty}^x f_X(t) \, dt 
         = \underbrace{\int_{-\infty}^0 f_X(t)dt}_{=F_X(0) = 0} + \int_0^x \frac{3t}{2} \, dt = \frac{3x^2}{4}

   **Region 3** (:math:`1 < x < 5`): No new probability, but keep previous accumulation

      .. math:: 
         F_X(x) &= \int_{-\infty}^x f_X(t) \, dt \\
         &= \underbrace{\int_{-\infty}^0 f_X(t)dt + \int_0^1 \frac{3t}{2} \, dt}_{=F_X(1) 
         = \frac{3}{4}} + \underbrace{\int_1^x f_X(t) \, dt}_{=0} = \frac{3}{4}
      
      The area is constant at the triangle's total area.

   **Region 4** (:math:`5 \leq x \leq 6`): Add rectangular area to previous accumulation

      .. math:: 
         F_X(x) &= \int_{-\infty}^x f_X(t) \, dt \\
         &= \underbrace{\int_{-\infty}^0 f_X(t)dt + \int_0^1 \frac{3t}{2} \, dt 
         + \int_1^5 f_X(t) \, dt}_{=F_X(5)=\frac{3}{4}} + \int_5^x f_X(t) \, dt\\
         &=\frac{3}{4} + \int_5^x \frac{1}{4} \, dt = \frac{3}{4} + \frac{x-5}{4} = \frac{x-2}{4}

   **Region 5** (:math:`x > 6`): All probability has been accumulated 

      .. math:: 
         F_X(x) &= \int_{-\infty}^x f_X(t) \, dt \\
         &= F_X(6) + \int_{6}^{\infty} f_X(t) \, dt = 1 + \int_6^{\infty} 0 \, dt = 1.

   Putting together,

   .. math::

      F_X(x) =
      \begin{cases}
      0 & \text{for } x < 0 \\
      \frac{3x^2}{4} & \text{for } 0 \leq x \leq 1 \\
      \frac{3}{4} & \text{for } 1 < x < 5 \\
      \frac{x-2}{4} & \text{for } 5 \leq x \leq 6 \\
      1 & \text{for } x > 6
      \end{cases}

.. admonition:: ðŸ›‘ Know how to avoid the common mistake before simplifying steps
   :class: danger

   In all regions beyond the first in the previous example, 
   the CDF :math:`f_X(x)` is computed as the sum of area accumulated in the earlier regions **plus** 
   the integral over the current region. 

   Forgetting to factor in the area from previous regions is a very common mistake made by students.
   While you may eventually skip some redundant steps as you gain familiarity, 
   your top priority should be to avoid this error.

Computing Probabilities with CDFs
------------------------------------

Once we have the CDF, calculating probabilities becomes remarkably straightforward. 
We can handle any probability problems without additional integration.

.. admonition:: Recall that equality signs do not matter for continuous RVs
   :class: important 

   All the rules below will apply if :math:`<` and :math:`\leq` are interchanged
   as well as if :math:`>` and :math:`\geq` are switched
   because :math:`P(X = a) = 0` for any single point :math:`a`.

Basic CDF Evaluation
~~~~~~~~~~~~~~~~~~~~~~~~

For :math:`P(X \leq a)`,

.. math::

   P(X \leq a) = F_X(a)

This is just the definition of the CDFâ€”plug in the value and read the result.

"Greater Than" Probabilities
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

For :math:`P(X > a)`, we use the complement rule:

.. math::

   P(X > a) = 1 - P(X \leq a) = 1 - F_X(a)

The probability of exceeding :math:`a` equals 
one minus the probability of not exceeding :math:`a`.

Interval Probabilities
~~~~~~~~~~~~~~~~~~~~~~~~

For :math:`P(a < X \leq b)`, we take the 
accumulated probability up to :math:`b` and **subtract** the accumulated probability 
up to :math:`a`, leaving just the probability in the interval.

.. math::

   P(a < X \leq b) = F_X(b) - F_X(a)

.. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/interval-prob-cdf.png
   :alt: Computing interval probabilities with CDFs
   :align: center
   :figwidth: 80%

   *P(a < X â‰¤ b) equals the difference between CDF values*

.. admonition:: ExampleðŸ’¡: Computing Probabilities Using a CDF
   :class: note 

   Use the PDF and CDF found in the previous example to answer the questions below.

   .. math::

      f_X(x) = \begin{cases}
      \frac{3x}{2} & \text{for } 0 \leq x \leq 1 \\
      \frac{1}{4} & \text{for } 5 \leq x \leq 6 \\
      0 & \text{elsewhere}
      \end{cases}

   .. math::

      F_X(x) = \begin{cases}
      0 & \text{for } x < 0 \\
      \frac{3x^2}{4} & \text{for } 0 \leq x < 1 \\
      \frac{3}{4} & \text{for } 1 \leq x < 5 \\
      \frac{x-2}{4} & \text{for } 5 \leq x \leq 6 \\
      1 & \text{for } x > 6
      \end{cases}

   1. Find :math:`P(X \leq 0.5)`.
      
      :math:`x=0.5` belongs to Region 2 of the CDF, so :math:`F_X(0.5) = \frac{3(0.5)^2}{4} = \frac{3}{16}`.

   2. Find :math:`P(X \geq 5.5)`.

      :math:`P(X \geq 5.5) = 1 - P(X < 5.5) = 1 - F_X(5.5)`. :math:`x=5.5` is in Region 4.
      Therefore, :math:`P(X > 5.5) = 1 - \frac{5.5-2}{4} = 1 - \frac{7}{8} = \frac{1}{8}`

   3. Find :math:`P(0.2 < X \leq 5.8)`.

      .. math:: 
         P(0.2 < X \leq 5.8) &= F_X(5.8) - F_X(0.2)
         = \frac{5.8-2}{4} -\frac{3(0.2)^2}{4}\\ &= 0.95 - 0.03 = 0.92.

Finding Percentiles using CDF
-----------------------------------

Definition
~~~~~~~~~~~~~

Take a value :math:`p \in [0,1]`. The :math:`p \times 100` th percentile of a 
random variable :math:`X`, denoted :math:`x_p`, is the value satisfying:

.. math::

   F_X(x_p) = P(X \leq x_p) = p.

For example,

* The **50th percentile** of a random variable :math:`X` is written as :math:`x_{0.5}`
  and satisfies the conditon:
  
  .. math:: F_X(x_{0.5}) = P(X \leq x_{0.5})= 0.5.
  
  :math:`x_{0.5}` represents the vertical cutoff on the PDF of :math:`X`
  such that the area to its left is exactly :math:`0.5`.
  This special percentile is also called the **median** (:math:`\tilde{\mu}`) of :math:`X`.

  .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/50th-percentile.png
   :alt: 50th percentile
   :align: center
   :figwidth: 60%

* The **25th percentile** :math:`x_{0.25}` satisfies :math:`F_X(x_{0.25}) = 0.25`.  
  It splits the PDF into a left region with area 0.25 and
  right region with area 0.75.

* The **75th percentile** :math:`x_{0.75}` satisfies :math:`F_X(x_{0.75}) = 0.75`.  
  It splits the PDF into a left region with area 0.75 and
  right region with area 0.25.

* The *true* IQR of :math:`X` is the difference of :math:`x_{0.75} - x_{0.25}`.

.. admonition:: Connection to *sample* percentiles
   :class: important 
   
   Recall that we've already discussed the concept of *sample* percentiles for
   a *dataset* in Chapter 3. The percentiles for a *random variable* are considered 
   their *population* equivalent in the framework of data analysis.  
   
   If we generate data points :math:`x_1, x_2,\cdots, x_n` from
   a random variable :math:`X`, we would compute the *sample* percentiles using 
   :math:`x_1, x_2,\cdots, x_n`, and the *true* percentiles using the CDF
   of :math:`X`. The two sets of objects are closely related, but different.

Computing Percentiles of a Continuous Random Variable
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Percentiles for a continuous random variables are found by **replacing the 
general definition** :math:`F_X(x_p) = p` **with specific expressions, than solving for** :math:`x_p`.
See the example below for further detail.

.. admonition:: ExampleðŸ’¡: Finding a Percentile
   :class: note 

   Find the 85th percentile of a random variable :math:`X` which has the
   following CDF:

   .. math::

      F_X(x) = \begin{cases}
      0 & \text{for } x < 0 \\
      \frac{3x^2}{4} & \text{for } 0 \leq x < 1 \\
      \frac{3}{4} & \text{for } 1 \leq x < 5 \\
      \frac{x-2}{4} & \text{for } 5 \leq x \leq 6 \\
      1 & \text{for } x > 6
      \end{cases}

   We must solve :math:`F_X(x_{0.85}) = 0.85` for :math:`x_{0.85}`.

   Which piece of the CDF should we use to replace the general expression :math:`F_X(x_{0.85})`?
   Since :math:`F_X(5) = \frac{3}{4} = 0.75 < 0.85`, we know that the 85th percentile has to be
   strictly greater than :math:`5`. Using Region 4, 

   .. math::
      &\frac{x_{0.85} - 2}{4} = 0.85\\
      &x_{0.85} - 2 = 3.4\\
      &x_{0.85} = 5.4

Bringing It All Together
---------------------------

.. admonition:: Key Takeaways ðŸ“
   :class: important

   1. The **cumulative distribution function** :math:`F_X(x) = P(X \leq x)` provides a universal framework for 
      both discrete and continuous random variables. It **eliminates the need for repeated integration** 
      when computing probabilities for continuous random variables.
   
   2. For continuous variables, **PDFs and CDFs are related** through differentiation and integration: 
      :math:`f_X(x) = F_X'(x)` and :math:`F_X(x) = \int_{-\infty}^x f_X(t) \, dt`.
   
   3. **Valid CDFs** must be non-decreasing, approach 0 as :math:`x \to -\infty`, approach 1 as :math:`x \to +\infty`, 
      and be right-continuous.
   
   4. **Probability calculations** become simpler with CDFs: :math:`P(X \leq a) = F_X(a)`, 
      :math:`P(X > a) = 1 - F_X(a)`, and :math:`P(a < X \leq b) = F_X(b) - F_X(a)`.
   
   5. **Piecewise CDFs** require careful attentionâ€”each region must 
      include all probability accumulated from previous regions.
   
   6. **Percentiles** are found by solving :math:`F_X(x_p) = p`.

In our next sections, we'll see how these concepts apply to specific named distributions, 
starting with the most important continuous distribution in all of statistics: the normal distribution.


Exercises
---------

These exercises develop your skills in constructing cumulative distribution functions, computing probabilities using CDFs, finding percentiles, and understanding the PDF-CDF relationship.

.. admonition:: Reminder
   :class: tip

   For continuous random variables, :math:`P(X = a) = 0` for any single point :math:`a`. This means strict and non-strict inequalities give the same probability: :math:`P(X < a) = P(X \leq a)` and :math:`P(X > a) = P(X \geq a)`. Throughout these exercises, we use whichever form is convenient.

.. admonition:: Exercise 1: Basic CDF Construction
   :class: note

   A quality control engineer models the thickness :math:`X` (in mm) of a protective coating with PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      2x, & 0 \leq x \leq 1\\
      0, & \text{elsewhere}
      \end{cases}

   a. Construct the CDF :math:`F_X(x)` for all regions.

   b. Verify that your CDF satisfies all three required properties: monotonicity, limiting behavior, and right-continuity.

   c. Use the CDF to find :math:`P(X \leq 0.6)`.

   d. Use the CDF to find :math:`P(X > 0.7)`.

   e. Use the CDF to find :math:`P(0.3 < X \leq 0.8)`.

   f. Find the median coating thickness.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Construct the CDF**

      **Region 1** (:math:`x < 0`): No probability accumulated yet.

      .. math::
         F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt = \int_{-\infty}^{x} 0 \, dt = 0

      **Region 2** (:math:`0 \leq x \leq 1`): Accumulate probability from the start of support.

      .. math::
         F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt = \underbrace{\int_{-\infty}^{0} 0 \, dt}_{=0} + \int_{0}^{x} 2t \, dt = t^2 \Big|_0^x = x^2

      **Region 3** (:math:`x > 1`): All probability accumulated.

      .. math::
         F_X(x) = F_X(1) + \int_{1}^{x} 0 \, dt = 1 + 0 = 1

      **Complete CDF**:

      .. math::

         F_X(x) = 
         \begin{cases}
         0, & x < 0\\
         x^2, & 0 \leq x \leq 1\\
         1, & x > 1
         \end{cases}

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig1_basic_cdf.png
         :alt: PDF and CDF for linear density
         :align: center
         :width: 80%

         Left: PDF :math:`f_X(x) = 2x`. Right: CDF :math:`F_X(x) = x^2` on [0,1].

      **Part (b): Verify CDF properties**

      **1. Monotonicity**: On :math:`[0, 1]`, :math:`F_X(x) = x^2` has derivative :math:`F_X'(x) = 2x \geq 0`, so it's non-decreasing. The function is constant (0 or 1) outside this interval. âœ“

      **2. Limiting behavior**:
      
      - :math:`\lim_{x \to -\infty} F_X(x) = 0` âœ“
      - :math:`\lim_{x \to +\infty} F_X(x) = 1` âœ“

      **3. Right-continuity**: The function is continuous everywhere (no jumps), so right-continuity is satisfied automatically. âœ“

      **Part (c): P(X â‰¤ 0.6)**

      .. math::
         P(X \leq 0.6) = F_X(0.6) = (0.6)^2 = 0.36

      **Part (d): P(X > 0.7)**

      .. math::
         P(X > 0.7) = 1 - P(X \leq 0.7) = 1 - F_X(0.7) = 1 - (0.7)^2 = 1 - 0.49 = 0.51

      **Part (e): P(0.3 < X â‰¤ 0.8)**

      .. math::
         P(0.3 < X \leq 0.8) = F_X(0.8) - F_X(0.3) = (0.8)^2 - (0.3)^2 = 0.64 - 0.09 = 0.55

      **Part (f): Median**

      Solve :math:`F_X(x_{0.5}) = 0.5`:

      .. math::
         x_{0.5}^2 = 0.5 \implies x_{0.5} = \sqrt{0.5} = \frac{1}{\sqrt{2}} \approx 0.707 \text{ mm}

.. admonition:: Exercise 2: CDF with Non-Zero Starting Point
   :class: note

   A biomedical engineer models drug concentration :math:`X` (in mg/L) in the bloodstream with PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{1}{2}(x - 2), & 2 \leq x \leq 4\\
      0, & \text{elsewhere}
      \end{cases}

   Note that the support starts at :math:`x = 2`, not :math:`x = 0`.

   a. Verify this is a valid PDF.

   b. Construct the complete CDF :math:`F_X(x)`.

   c. Find :math:`P(X \leq 3)`.

   d. Find :math:`P(2.5 < X < 3.5)`.

   e. Find the 75th percentile of drug concentration.

   f. **Common Mistake Alert**: A student writes :math:`F_X(3) = \int_0^3 f_X(x) \, dx`. Explain the error.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF validity**

      **Non-negativity**: On :math:`[2, 4]`, :math:`x - 2 \geq 0`, so :math:`f_X(x) = \frac{1}{2}(x-2) \geq 0`. âœ“

      **Total area = 1**:

      .. math::
         \int_{2}^{4} \frac{1}{2}(x - 2) \, dx = \frac{1}{2} \cdot \frac{(x-2)^2}{2} \Bigg|_{2}^{4} = \frac{1}{4}[(4-2)^2 - 0] = \frac{1}{4} \cdot 4 = 1 \text{ âœ“}

      **Part (b): Construct CDF**

      **Region 1** (:math:`x < 2`):

      .. math::
         F_X(x) = 0

      **Region 2** (:math:`2 \leq x \leq 4`):

      .. math::
         F_X(x) = \int_{2}^{x} \frac{1}{2}(t - 2) \, dt = \frac{1}{2} \cdot \frac{(t-2)^2}{2} \Bigg|_{2}^{x} = \frac{(x-2)^2}{4}

      **Region 3** (:math:`x > 4`):

      .. math::
         F_X(x) = 1

      **Complete CDF**:

      .. math::

         F_X(x) = 
         \begin{cases}
         0, & x < 2\\
         \frac{(x-2)^2}{4}, & 2 \leq x \leq 4\\
         1, & x > 4
         \end{cases}

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig2_nonzero_start_cdf.png
         :alt: PDF and CDF with support starting at x=2
         :align: center
         :width: 80%

         The CDF starts accumulating at :math:`x = 2`, not :math:`x = 0`.

      **Part (c): P(X â‰¤ 3)**

      .. math::
         P(X \leq 3) = F_X(3) = \frac{(3-2)^2}{4} = \frac{1}{4} = 0.25

      **Part (d): P(2.5 < X < 3.5)**

      .. math::
         P(2.5 < X < 3.5) = F_X(3.5) - F_X(2.5) = \frac{(1.5)^2}{4} - \frac{(0.5)^2}{4} = \frac{2.25 - 0.25}{4} = \frac{2}{4} = 0.5

      **Part (e): 75th percentile**

      Solve :math:`F_X(x_{0.75}) = 0.75`:

      .. math::
         \frac{(x_{0.75} - 2)^2}{4} = 0.75

      .. math::
         (x_{0.75} - 2)^2 = 3

      .. math::
         x_{0.75} = 2 + \sqrt{3} \approx 3.732 \text{ mg/L}

      **Part (f): Common Mistake**

      The student integrated from 0, but the PDF equals 0 for :math:`x < 2`. The CDF is defined as:

      .. math::
         F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt

      Since :math:`f_X(t) = 0` for :math:`t < 2`, the integral from :math:`-\infty` to 2 contributes nothing. The correct calculation integrates only over the support where :math:`f_X > 0`.

.. admonition:: Exercise 3: Piecewise CDF Construction
   :class: note

   A computer scientist models network latency :math:`X` (in milliseconds) with PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{x}{3}, & 0 \leq x \leq 2\\
      \frac{1}{3}, & 2 < x \leq 3\\
      0, & \text{elsewhere}
      \end{cases}

   a. Verify this is a valid PDF.

   b. Construct the complete piecewise CDF.

   c. Use the CDF to find :math:`P(1 < X \leq 2.5)`.

   d. Find the median latency.

   e. Find the 90th percentile.

   f. Find the interquartile range (IQR).

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF validity**

      **Non-negativity**: Both pieces are non-negative on their domains. âœ“

      **Total area = 1**:

      .. math::
         \int_{0}^{2} \frac{x}{3} \, dx + \int_{2}^{3} \frac{1}{3} \, dx = \frac{1}{3} \cdot \frac{x^2}{2}\Bigg|_0^2 + \frac{1}{3} \cdot 1 = \frac{1}{3} \cdot 2 + \frac{1}{3} = \frac{2}{3} + \frac{1}{3} = 1 \text{ âœ“}

      **Part (b): Construct piecewise CDF**

      **Region 1** (:math:`x < 0`):

      .. math::
         F_X(x) = 0

      **Region 2** (:math:`0 \leq x \leq 2`):

      .. math::
         F_X(x) = \int_{0}^{x} \frac{t}{3} \, dt = \frac{x^2}{6}

      Check: :math:`F_X(2) = \frac{4}{6} = \frac{2}{3}`

      **Region 3** (:math:`2 < x \leq 3`):

      .. math::
         F_X(x) = F_X(2) + \int_{2}^{x} \frac{1}{3} \, dt = \frac{2}{3} + \frac{x-2}{3} = \frac{x}{3}

      Check: :math:`F_X(3) = 1` âœ“

      **Region 4** (:math:`x > 3`):

      .. math::
         F_X(x) = 1

      **Complete CDF**:

      .. math::

         F_X(x) = 
         \begin{cases}
         0, & x < 0\\
         \frac{x^2}{6}, & 0 \leq x \leq 2\\
         \frac{x}{3}, & 2 < x \leq 3\\
         1, & x > 3
         \end{cases}

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig3_piecewise_cdf.png
         :alt: Piecewise PDF and its CDF
         :align: center
         :width: 80%

         Piecewise PDF (left) and its CDF (right). Note the CDF is continuous but has different formulas in each region.

      **Part (c): P(1 < X â‰¤ 2.5)**

      .. math::
         P(1 < X \leq 2.5) = F_X(2.5) - F_X(1) = \frac{2.5}{3} - \frac{1}{6} = \frac{5}{6} - \frac{1}{6} = \frac{4}{6} = \frac{2}{3}

      **Part (d): Median**

      We need :math:`F_X(x_{0.5}) = 0.5`.

      Since :math:`F_X(2) = \frac{2}{3} > 0.5`, the median is in Region 2.

      .. math::
         \frac{x_{0.5}^2}{6} = 0.5 \implies x_{0.5}^2 = 3 \implies x_{0.5} = \sqrt{3} \approx 1.732 \text{ ms}

      **Part (e): 90th percentile**

      Since :math:`F_X(2) = \frac{2}{3} \approx 0.667 < 0.9`, the 90th percentile is in Region 3.

      .. math::
         \frac{x_{0.9}}{3} = 0.9 \implies x_{0.9} = 2.7 \text{ ms}

      **Part (f): IQR**

      **25th percentile**: :math:`F_X(2) = 0.667 > 0.25`, so use Region 2:

      .. math::
         \frac{x_{0.25}^2}{6} = 0.25 \implies x_{0.25}^2 = 1.5 \implies x_{0.25} = \sqrt{1.5} \approx 1.225 \text{ ms}

      **75th percentile**: :math:`F_X(2) = 0.667 < 0.75`, so use Region 3:

      .. math::
         \frac{x_{0.75}}{3} = 0.75 \implies x_{0.75} = 2.25 \text{ ms}

      **IQR**:

      .. math::
         \text{IQR} = x_{0.75} - x_{0.25} = 2.25 - 1.225 = 1.025 \text{ ms}

.. admonition:: Exercise 4: Piecewise CDF with Gap in Support
   :class: note

   An industrial engineer models machine cycle time :math:`X` (in seconds) with PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{1}{3}, & 0 \leq x \leq 2\\
      \frac{1}{6}(x - 4), & 4 \leq x \leq 6\\
      0, & \text{elsewhere}
      \end{cases}

   Note the **gap** in support from :math:`x = 2` to :math:`x = 4`.

   a. Verify this is a valid PDF.

   b. Construct the complete piecewise CDF, including the gap region.

   c. Find :math:`P(1 < X < 5)`.

   d. Find the median.

   e. If a cycle takes more than 4 seconds (slow mode), what is :math:`P(X > 5 | X > 4)`?

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF validity**

      **Non-negativity**: Region 1 has constant positive density. Region 2 has :math:`x - 4 \geq 0` for :math:`x \geq 4`. âœ“

      **Total area = 1**:

      .. math::
         \int_{0}^{2} \frac{1}{3} \, dx + \int_{4}^{6} \frac{1}{6}(x - 4) \, dx = \frac{2}{3} + \frac{1}{6} \cdot \frac{(x-4)^2}{2}\Bigg|_4^6

      .. math::
         = \frac{2}{3} + \frac{1}{12} \cdot 4 = \frac{2}{3} + \frac{1}{3} = 1 \text{ âœ“}

      **Part (b): Construct piecewise CDF**

      **Region 1** (:math:`x < 0`): :math:`F_X(x) = 0`

      **Region 2** (:math:`0 \leq x \leq 2`):

      .. math::
         F_X(x) = \int_0^x \frac{1}{3} \, dt = \frac{x}{3}

      :math:`F_X(2) = \frac{2}{3}`

      **Region 3** (:math:`2 < x < 4`): **Gap region** â€” no new probability accumulates

      .. math::
         F_X(x) = F_X(2) = \frac{2}{3}

      Note: :math:`F_X(4) = \frac{2}{3}` as well (the CDF remains constant through the gap).

      **Region 4** (:math:`4 \leq x \leq 6`):

      .. math::
         F_X(x) = F_X(4) + \int_4^x \frac{1}{6}(t-4) \, dt = \frac{2}{3} + \frac{(x-4)^2}{12}

      :math:`F_X(6) = \frac{2}{3} + \frac{4}{12} = \frac{2}{3} + \frac{1}{3} = 1` âœ“

      **Region 5** (:math:`x > 6`): :math:`F_X(x) = 1`

      **Complete CDF**:

      .. math::

         F_X(x) = 
         \begin{cases}
         0, & x < 0\\
         \frac{x}{3}, & 0 \leq x \leq 2\\
         \frac{2}{3}, & 2 < x < 4\\
         \frac{2}{3} + \frac{(x-4)^2}{12}, & 4 \leq x \leq 6\\
         1, & x > 6
         \end{cases}

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig4_gap_cdf.png
         :alt: PDF with gap and its CDF showing flat region
         :align: center
         :width: 80%

         The CDF is flat (horizontal) during the gap where the PDF is zero.

      **Part (c): P(1 < X < 5)**

      .. math::
         P(1 < X < 5) = F_X(5) - F_X(1) = \left[\frac{2}{3} + \frac{1}{12}\right] - \frac{1}{3} = \frac{3}{4} - \frac{1}{3} = \frac{9-4}{12} = \frac{5}{12}

      **Part (d): Median**

      Since :math:`F_X(2) = \frac{2}{3} > 0.5`, the median is in Region 2.

      .. math::
         \frac{x_{0.5}}{3} = 0.5 \implies x_{0.5} = 1.5 \text{ seconds}

      **Part (e): Conditional probability**

      .. math::
         P(X > 5 | X > 4) = \frac{P(X > 5)}{P(X > 4)} = \frac{1 - F_X(5)}{1 - F_X(4)}

      .. math::
         = \frac{1 - \frac{3}{4}}{1 - \frac{2}{3}} = \frac{1/4}{1/3} = \frac{3}{4}

.. admonition:: Exercise 5: From CDF to PDF (Differentiation)
   :class: note

   A materials engineer is given the following CDF for material strength :math:`X` (in MPa):

   .. math::

      F_X(x) = 
      \begin{cases}
      0, & x < 0\\
      \frac{x^3}{8}, & 0 \leq x \leq 2\\
      1, & x > 2
      \end{cases}

   a. Find the PDF :math:`f_X(x)` by differentiating the CDF.

   b. Verify your PDF integrates to 1.

   c. Find :math:`P(X > 1.5)`.

   d. Find the median strength.

   e. Find the 25th and 75th percentiles, then compute the IQR.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Find PDF by differentiation**

      The PDF is the derivative of the CDF: :math:`f_X(x) = \frac{d}{dx}F_X(x)`.

      **Region** :math:`0 \leq x \leq 2`:

      .. math::
         f_X(x) = \frac{d}{dx}\left(\frac{x^3}{8}\right) = \frac{3x^2}{8}

      **Complete PDF**:

      .. math::

         f_X(x) = 
         \begin{cases}
         \frac{3x^2}{8}, & 0 \leq x \leq 2\\
         0, & \text{elsewhere}
         \end{cases}

      **Part (b): Verify PDF integrates to 1**

      .. math::
         \int_0^2 \frac{3x^2}{8} \, dx = \frac{3}{8} \cdot \frac{x^3}{3}\Bigg|_0^2 = \frac{1}{8} \cdot 8 = 1 \text{ âœ“}

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig5_cdf_to_pdf.png
         :alt: CDF and its derivative (PDF)
         :align: center
         :width: 80%

         The PDF is the derivative of the CDF. Where the CDF rises steeply, the PDF is large.

      **Part (c): P(X > 1.5)**

      .. math::
         P(X > 1.5) = 1 - F_X(1.5) = 1 - \frac{(1.5)^3}{8} = 1 - \frac{3.375}{8} = 1 - 0.422 = 0.578

      **Part (d): Median**

      Solve :math:`F_X(x_{0.5}) = 0.5`:

      .. math::
         \frac{x_{0.5}^3}{8} = 0.5 \implies x_{0.5}^3 = 4 \implies x_{0.5} = \sqrt[3]{4} \approx 1.587 \text{ MPa}

      **Part (e): Quartiles and IQR**

      **25th percentile**:

      .. math::
         \frac{x_{0.25}^3}{8} = 0.25 \implies x_{0.25}^3 = 2 \implies x_{0.25} = \sqrt[3]{2} \approx 1.260 \text{ MPa}

      **75th percentile**:

      .. math::
         \frac{x_{0.75}^3}{8} = 0.75 \implies x_{0.75}^3 = 6 \implies x_{0.75} = \sqrt[3]{6} \approx 1.817 \text{ MPa}

      **IQR**:

      .. math::
         \text{IQR} = x_{0.75} - x_{0.25} = \sqrt[3]{6} - \sqrt[3]{2} \approx 1.817 - 1.260 = 0.557 \text{ MPa}

.. admonition:: Exercise 6: Exponential CDF
   :class: note

   A reliability engineer models time-to-failure :math:`X` (in hours) for an electronic component with CDF:

   .. math::

      F_X(x) = 
      \begin{cases}
      0, & x < 0\\
      1 - e^{-x/100}, & x \geq 0
      \end{cases}

   a. Find the PDF by differentiation.

   b. Find :math:`P(X > 150)`, the probability a component lasts more than 150 hours.

   c. Find the median lifetime.

   d. Find the 95th percentile (often used as a "design life" specification).

   e. If a component has already survived 50 hours, find :math:`P(X > 150 | X > 50)`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Find PDF**

      .. math::
         f_X(x) = \frac{d}{dx}\left(1 - e^{-x/100}\right) = \frac{1}{100}e^{-x/100} \text{ for } x \geq 0

      This is an **exponential distribution** with rate :math:`\lambda = \frac{1}{100}` (or mean :math:`\mu = 100` hours).

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig6_exponential_cdf.png
         :alt: Exponential PDF and CDF
         :align: center
         :width: 80%

         Exponential distribution: PDF decays exponentially; CDF approaches 1 asymptotically.

      **Part (b): P(X > 150)**

      .. math::
         P(X > 150) = 1 - F_X(150) = 1 - (1 - e^{-150/100}) = e^{-1.5} \approx 0.223

      **Part (c): Median**

      Solve :math:`F_X(x_{0.5}) = 0.5`:

      .. math::
         1 - e^{-x_{0.5}/100} = 0.5 \implies e^{-x_{0.5}/100} = 0.5

      .. math::
         -\frac{x_{0.5}}{100} = \ln(0.5) = -\ln(2) \implies x_{0.5} = 100\ln(2) \approx 69.3 \text{ hours}

      **Part (d): 95th percentile**

      Solve :math:`F_X(x_{0.95}) = 0.95`:

      .. math::
         1 - e^{-x_{0.95}/100} = 0.95 \implies e^{-x_{0.95}/100} = 0.05

      .. math::
         -\frac{x_{0.95}}{100} = \ln(0.05) \implies x_{0.95} = -100\ln(0.05) = 100\ln(20) \approx 299.6 \text{ hours}

      **Part (e): Memoryless property**

      For exponential distributions:

      .. math::
         P(X > 150 | X > 50) = P(X > 100) = e^{-100/100} = e^{-1} \approx 0.368

      The probability of surviving an additional 100 hours is the same regardless of how long the component has already lasted. This is the **memoryless property** unique to exponential distributions.

.. admonition:: Exercise 7: Verifying CDF Validity
   :class: note

   Determine whether each of the following could be a valid CDF. If valid, find the corresponding PDF. If not valid, explain which property is violated.

   a. :math:`F(x) = \begin{cases} 0, & x < 0\\ x^2, & 0 \leq x \leq 1\\ 1, & x > 1 \end{cases}`

   b. :math:`F(x) = \begin{cases} 0, & x < 0\\ 2x - x^2, & 0 \leq x \leq 1\\ 1, & x > 1 \end{cases}`

   c. :math:`F(x) = \begin{cases} 0, & x < 1\\ \frac{x-1}{2}, & 1 \leq x \leq 3\\ 1, & x > 3 \end{cases}`

   d. :math:`F(x) = \begin{cases} 0, & x < 0\\ x^3 - x^2, & 0 \leq x \leq 1\\ 1, & x > 1 \end{cases}`

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Valid CDF**

      Check properties:
      
      - **Monotonicity**: :math:`F'(x) = 2x \geq 0` for :math:`x \in [0,1]`. âœ“
      - **Limits**: :math:`F(-\infty) = 0`, :math:`F(\infty) = 1`. âœ“
      - **Right-continuous**: Continuous everywhere. âœ“
      - **Boundary values**: :math:`F(0) = 0`, :math:`F(1) = 1`. âœ“

      **PDF**: :math:`f(x) = 2x` for :math:`0 \leq x \leq 1`, 0 elsewhere.

      **Part (b): Valid CDF**

      Check properties:
      
      - **Monotonicity**: :math:`F'(x) = 2 - 2x = 2(1-x) \geq 0` for :math:`x \in [0,1]`. âœ“
      - **Limits**: :math:`F(-\infty) = 0`, :math:`F(\infty) = 1`. âœ“
      - **Boundary values**: :math:`F(0) = 0`, :math:`F(1) = 2 - 1 = 1`. âœ“

      **PDF**: :math:`f(x) = 2 - 2x = 2(1-x)` for :math:`0 \leq x \leq 1`, 0 elsewhere.

      **Part (c): Valid CDF**

      Check properties:
      
      - **Monotonicity**: :math:`F'(x) = \frac{1}{2} > 0`. âœ“
      - **Limits**: âœ“
      - **Boundary values**: :math:`F(1) = 0`, :math:`F(3) = 1`. âœ“

      **PDF**: :math:`f(x) = \frac{1}{2}` for :math:`1 \leq x \leq 3`, 0 elsewhere. (Uniform distribution on [1, 3])

      **Part (d): NOT a valid CDF**

      Check monotonicity: :math:`F'(x) = 3x^2 - 2x = x(3x - 2)`.

      For :math:`0 < x < \frac{2}{3}`, we have :math:`3x - 2 < 0`, so :math:`F'(x) < 0`.

      **Violations**:

      1. **Negative values**: On :math:`(0, 1)`, :math:`F(x) = x^3 - x^2 = x^2(x-1) \leq 0` since :math:`x - 1 < 0`. A CDF must satisfy :math:`0 \leq F(x) \leq 1` for all :math:`x`.

      2. **Not nondecreasing**: The derivative is negative on :math:`(0, \frac{2}{3})`, so the function decreases on part of :math:`[0,1]`.

      3. **Right-continuity failure at** :math:`x = 1`: With the piecewise definition, :math:`F(1) = 1^3 - 1^2 = 0` from the middle piece, but :math:`\lim_{x \to 1^+} F(x) = 1` from the right piece. Since :math:`F(1) \neq \lim_{x \to 1^+} F(x)`, right-continuity fails.

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig7_cdf_validity.png
         :alt: Valid vs invalid CDFs
         :align: center
         :width: 80%

         Top row: Valid CDFs are non-decreasing. Bottom: Invalid CDF decreases in the middle.

.. admonition:: Exercise 8: Symmetric Distribution Percentiles
   :class: note

   A sensor measurement error :math:`X` (in Î¼V) has the symmetric triangular PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{1}{4}(x + 2), & -2 \leq x \leq 0\\
      \frac{1}{4}(2 - x), & 0 < x \leq 2\\
      0, & \text{elsewhere}
      \end{cases}

   a. Verify this is a valid PDF.

   b. Use symmetry to determine the median without calculation.

   c. Construct the CDF.

   d. Find the 10th and 90th percentiles.

   e. Verify that :math:`x_{0.1}` and :math:`x_{0.9}` are symmetric about the median.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF**

      The PDF forms a triangle with base 4 (from -2 to 2) and height :math:`\frac{1}{2}` (at x = 0).

      Area = :math:`\frac{1}{2} \times 4 \times \frac{1}{2} = 1` âœ“

      **Part (b): Median by symmetry**

      The PDF is symmetric about :math:`x = 0`. By symmetry, the median is:

      .. math::
         x_{0.5} = 0 \text{ Î¼V}

      **Part (c): Construct CDF**

      **Region 1** (:math:`x < -2`): :math:`F_X(x) = 0`

      **Region 2** (:math:`-2 \leq x \leq 0`):

      .. math::
         F_X(x) = \int_{-2}^{x} \frac{1}{4}(t + 2) \, dt = \frac{1}{4} \cdot \frac{(t+2)^2}{2}\Bigg|_{-2}^x = \frac{(x+2)^2}{8}

      :math:`F_X(0) = \frac{4}{8} = \frac{1}{2}` âœ“ (confirms median at 0)

      **Region 3** (:math:`0 < x \leq 2`):

      .. math::
         F_X(x) = \frac{1}{2} + \int_{0}^{x} \frac{1}{4}(2 - t) \, dt = \frac{1}{2} + \frac{1}{4}\left[2t - \frac{t^2}{2}\right]_0^x = \frac{1}{2} + \frac{x}{2} - \frac{x^2}{8}

      Or equivalently: :math:`F_X(x) = 1 - \frac{(2-x)^2}{8}` (by symmetry)

      **Region 4** (:math:`x > 2`): :math:`F_X(x) = 1`

      **Complete CDF**:

      .. math::

         F_X(x) = 
         \begin{cases}
         0, & x < -2\\
         \frac{(x+2)^2}{8}, & -2 \leq x \leq 0\\
         1 - \frac{(2-x)^2}{8}, & 0 < x \leq 2\\
         1, & x > 2
         \end{cases}

      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-3/fig8_symmetric_percentiles.png
         :alt: Symmetric triangular distribution with percentiles
         :align: center
         :width: 80%

         Symmetric triangular PDF with 10th and 90th percentiles marked equidistant from the median.

      **Part (d): 10th and 90th percentiles**

      **10th percentile** (in Region 2 since :math:`0.1 < 0.5`):

      .. math::
         \frac{(x_{0.1} + 2)^2}{8} = 0.1 \implies (x_{0.1} + 2)^2 = 0.8

      .. math::
         x_{0.1} = -2 + \sqrt{0.8} = -2 + 0.894 \approx -1.106 \text{ Î¼V}

      **90th percentile** (by symmetry, or using Region 3):

      .. math::
         1 - \frac{(2 - x_{0.9})^2}{8} = 0.9 \implies (2 - x_{0.9})^2 = 0.8

      .. math::
         x_{0.9} = 2 - \sqrt{0.8} \approx 1.106 \text{ Î¼V}

      **Part (e): Verify symmetry**

      We show :math:`x_{0.1} = -x_{0.9}`:

      .. math::
         x_{0.1} = -2 + \sqrt{0.8} = -(2 - \sqrt{0.8}) = -x_{0.9}

      Therefore the distances from the median (0) are equal:

      .. math::
         |x_{0.1} - 0| = |x_{0.9} - 0| = 2 - \sqrt{0.8} \approx 1.106 \text{ âœ“}

.. admonition:: Exercise 9: Working Backwards from Percentiles
   :class: note

   A manufacturing process produces components whose length :math:`X` (in cm) follows a distribution with CDF:

   .. math::

      F_X(x) = 
      \begin{cases}
      0, & x < a\\
      \left(\frac{x - a}{b - a}\right)^2, & a \leq x \leq b\\
      1, & x > b
      \end{cases}

   The quality control team has determined that:
   
   - The 25th percentile is 12 cm
   - The 75th percentile is 16 cm

   a. Set up a system of equations using these percentile conditions.

   b. Solve for the parameters :math:`a` and :math:`b`.

   c. Find the median length.

   d. Find the PDF.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): System of equations**

      Using :math:`F_X(x_{0.25}) = 0.25` and :math:`F_X(x_{0.75}) = 0.75`:

      .. math::
         \left(\frac{12 - a}{b - a}\right)^2 = 0.25 \implies \frac{12 - a}{b - a} = 0.5

      .. math::
         \left(\frac{16 - a}{b - a}\right)^2 = 0.75 \implies \frac{16 - a}{b - a} = \sqrt{0.75} = \frac{\sqrt{3}}{2}

      **Part (b): Solve for a and b**

      From equation 1: :math:`12 - a = 0.5(b - a)`, so :math:`24 - 2a = b - a`, giving :math:`b = 24 - a`.

      Substitute into equation 2:

      .. math::
         \frac{16 - a}{24 - 2a} = \frac{\sqrt{3}}{2}

      .. math::
         2(16 - a) = \sqrt{3}(24 - 2a)

      .. math::
         32 - 2a = 24\sqrt{3} - 2\sqrt{3}a

      .. math::
         32 - 24\sqrt{3} = 2a - 2\sqrt{3}a = 2a(1 - \sqrt{3})

      .. math::
         a = \frac{32 - 24\sqrt{3}}{2(1 - \sqrt{3})} = \frac{16 - 12\sqrt{3}}{1 - \sqrt{3}}

      Rationalizing: multiply by :math:`\frac{1 + \sqrt{3}}{1 + \sqrt{3}}`:

      .. math::
         a = \frac{(16 - 12\sqrt{3})(1 + \sqrt{3})}{(1 - \sqrt{3})(1 + \sqrt{3})} = \frac{16 + 16\sqrt{3} - 12\sqrt{3} - 36}{1 - 3}

      .. math::
         = \frac{-20 + 4\sqrt{3}}{-2} = 10 - 2\sqrt{3} \approx 6.54 \text{ cm}

      Then: :math:`b = 24 - a = 24 - 10 + 2\sqrt{3} = 14 + 2\sqrt{3} \approx 17.46 \text{ cm}`

      **Part (c): Median**

      .. math::
         \left(\frac{x_{0.5} - a}{b - a}\right)^2 = 0.5 \implies \frac{x_{0.5} - a}{b - a} = \frac{1}{\sqrt{2}}

      .. math::
         x_{0.5} = a + \frac{b - a}{\sqrt{2}} = (10 - 2\sqrt{3}) + \frac{4 + 4\sqrt{3}}{\sqrt{2}}

      .. math::
         = 10 - 2\sqrt{3} + 2\sqrt{2} + 2\sqrt{6} \approx 14.23 \text{ cm}

      **Part (d): PDF**

      .. math::
         f_X(x) = \frac{d}{dx}\left[\left(\frac{x - a}{b - a}\right)^2\right] = \frac{2(x - a)}{(b - a)^2}

      With :math:`b - a = 4 + 4\sqrt{3}`:

      .. math::
         f_X(x) = \frac{2(x - a)}{(4 + 4\sqrt{3})^2} = \frac{x - a}{8(1 + \sqrt{3})^2} \text{ for } a \leq x \leq b

   **True/False Answers:**

   1. **True** â€” For continuous random variables, :math:`P(X = a) = 0`, so :math:`P(X < a) = P(X \leq a)`.

   2. **True** â€” By definition, :math:`F_X(x) = P(X \leq x)`, and all probabilities are at most 1.

   3. **True** â€” :math:`P(X > 5) = 1 - P(X \leq 5) = 1 - F_X(5) = 1 - 0.7 = 0.3`.

   4. **False** â€” The PDF is obtained by **differentiation**: :math:`f_X(x) = F_X'(x)`. The CDF is obtained from the PDF by integration.

   5. **False** â€” The CDF is flat (constant) where the PDF is zero, but it retains the accumulated probability from previous regions. It only equals zero before the start of support.

   6. **True** â€” By definition, the :math:`p`-th percentile is :math:`x_p` where :math:`F_X(x_p) = p`. The median is the 50th percentile.

   7. **True** â€” CDFs are non-decreasing, so :math:`b > a` implies :math:`F_X(b) \geq F_X(a)`.

   8. **True** â€” For symmetric distributions, the median equals the center of symmetry.
