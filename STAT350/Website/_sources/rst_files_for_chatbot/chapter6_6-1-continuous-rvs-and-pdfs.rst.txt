Continuous Random Variables and Probability Density Functions
==================================================================================

What happens when we want to model **measurements** rather than counts? 
How do we handle quantities like height, weight, temperature, or timeâ€”variables that can take on any 
value within a continuous range? This section explores the shift in mathematical framework that occurs when 
we move from the discrete world of "**how many**" to the continuous world of "**how much**."


.. admonition:: Road Map ðŸ§­
   :class: important

   â€¢ Understand why **continuous random variables** require a different mathematical approach than discrete ones.
   â€¢ Define **probability density functions (PDFs)** as the continuous analog to probability mass functions.
   â€¢ Master the key concept that **probabilities are areas** under a PDF.
   â€¢ Learn the **essential properties** that make a function a valid PDF.
   â€¢ Find probabilities for continuous random variables by integrating the PDF.


Discrete vs. Continuous Random Variables
-----------------------------------------------------------

For random variables with a discrete support, we could assign **positive probabilities to individual outcomes**. 
It made perfect sense to say "the probability of getting exactly 3 heads in 10 coin flips 
is some specific value" because 3 was one of only eleven distinct outcomes (0 through 10 heads).
Even when the support is **countably infinite**â€”as in the Poisson distributionâ€”we could still assign 
probabilities in such a way that each value in the support had a positive probability (however small) while the
total sum remains 1.

But many real-world phenomena involve measurements along a continuous scale, which has a vastly larger support
than that of any discrete random variable. While we might record a person's height as "5 feet 8 inches," 
the actual height could be 5.75000... feet or 5.750001... feet. Between any two measurements, no matter how close, 
there are **uncountably many** possible intermediate values. 
If we tried to assign positive probabilities to each possible heightâ€”no matter how cleverlyâ€”we would 
end up with an infinite total, violating the fundamental requirement that all probabilities sum (or integrate) to 1.

The Resolution: Zero Probability for Any Single Point
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

We resolve this paradox by accepting that **any single exact 
value has probability zero** for continuous random variables. 
The probability that someone's height is exactly 5.750000000... 
feet with infinite precision is zero, even though this height is perfectly possible.

This might seem counterintuitive at first. How can something be possible but have zero probability? 
Recall that probability can be seen as the relative size of an event compared to the whole. 
In the continuous case, we're dealing with uncountably many possible values packed into any interval, 
so many that any single point is negligible in comparison. This makes the
*relative size*, and thus the probability, of any one value equal to zero. 

Then What Has a Positive Probability?
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

For continuous random variables, we discuss probabilities of *intervals* of values
rather than single points. This aligns naturally with the graphical interpretation of 
probabilities as areas under a curve--regions with non-zero width will have a
positive area, while a single point always has zero area.

Probability Density Functions: The Continuous Analog of Probability Mass Functions
-------------------------------------------------------------------------------------------------

From Histograms to Curves
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Since we can't assign probabilities to individual points, we need a different approach than PMF to
describe the distribution of a continuous random variable. 
We can think of a continuous probability distribution as a curve that represents the 
**limiting behavior of increasingly fine histograms for an increasingly large dataset**.

In :numref:`hist-to-pdf`, the jagged histogram begins to approximate a curve 
as we collect more data and make the bins narrower. In the 
limitâ€”with infinite data and infinitesimally narrow binsâ€”we get a 
**probability density function**.

   :alt: Evolution from histogram to probability density function
   :align: center
   :figwidth: 90%

   Evolution from histogram to a probability density function

Mathematical Definition
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

A **probability density function (PDF)** for a continuous random variable :math:`X`, 
denoted :math:`f_X(x)`, specifies how "dense" the probability is around each point. 

Mathematically,

.. math::

   f_X(x) = \lim_{\Delta \to 0^+} \frac{P(x < X \leq x + \Delta)}{\Delta}.

Interpreting a PDF
~~~~~~~~~~~~~~~~~~~~~

It is important to note that :math:`f_X` evaluated at any point :math:`x` 
tells us about the **relative** likelihood of values 
in that neighborhood. Suppose a random variable :math:`X` gives
:math:`f_X(5.8) = 3` and :math:`f_X(6.2) = 1`. We observe that:

* Values in a small neighborhood around 5.8 is three times more likely to occur
  than values in the neighborhood of 6.2.
* :math:`f_X` does NOT give probabilities directly. :math:`f_X(6.2) = 1` does NOT
  mean that the exact value of 6.2 occurs with probability 1.
* Evaluations of :math:`f_X` are not restricted 
  to be at most 1. The set of rules for validity of a PDF will be discussed below.

Support
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/support.png
   :alt: Support shown on a pdf graph
   :align: center
   :figwidth: 60%

The **support** of a continuous random variable :math:`X` is the set of all 
possible values that :math:`X` can take, or equivalently, the set of values 
where its PDF is strictly positive:

.. math::

   \text{supp}(X) = \{x \in \mathbb{R} \mid f_X(x) > 0\}.

Computing Probabilities: Areas Under the PDF
-----------------------------------------------------

.. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/prob-a-b.png
   :alt: Probability between a and b on a PDF
   :align: center
   :figwidth: 60%

For a continuous random variable :math:`X` with PDF :math:`f_X(x)`, the probability 
that :math:`X` takes a value between :math:`a` and :math:`b` is equal to the area under :math:`f_X(x)`
between points :math:`a` and :math:`b`. Mathematically,

.. math::

   P(a \leq X \leq b) = \int_a^b f_X(x) \, dx

Special Case: Probability at a Single Point
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/fx-not-prob.png
   :alt: Probability of a single point
   :align: center
   :figwidth: 60%

For any specific value :math:`a`,

.. math::

   P(X = a) = P(a \leq X \leq a) = \int_a^a f_X(x) \, dx = 0

Any integral from a point to itself is zero because the interval has zero width. 
Note that :math:`f_X(a)` evaluated at any :math:`a` in the support is positive.
This again highlights the fact that evaluating :math:`f_X` at a point does **not** directly 
give its probability.

An Important Consequence: Equality Doesn't Matter
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Since any single point has zero probability, using strict or weak inequalities does not affect the probabilities:

.. math::

   P(a \leq X \leq b) = P(a \leq X < b) = P(a < X \leq b) = P(a < X < b)

This is a major difference from discrete random variables, where :math:`P(X = k)` could be positive, 
making the choice between :math:`<` and :math:`\leq` crucial.

Properties of A Valid Probability Density Function
---------------------------------------------------------

Not every function can serve as a PDF. A valid PDF must satisfy two essential properties 
that parallel those required of discrete probability mass functions.

Property 1: Non-Negativity
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

A PDF must be non-negative everywhere. That is,

.. math::

   f_X(x) \geq 0 \text{ for all } x \in \mathbb{R}.

This makes senseâ€”there cannot be a likelihood smaller than none.
However, unlike discrete PMFs, PDFs are not constrained to values less than or equal to 1. 
They can take arbitrarily large values at some points, as long as they satisfy
the next property.

Property 2: Total Area Equals One
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The total area under a PDF must equal 1, or equivalently,

.. math::

   \int_{-\infty}^{\infty} f_X(x) \, dx = 1.

This implies that the probability of observing some value within the support is equal to 1.

.. admonition:: ExampleðŸ’¡: Validating and Working with a Simple PDF
   :class: note

   Suppose **the maximum diameter of a potato chip** (:math:`X`) produced at Factory A, in inches, follows
   the following probability density:

   .. math::
      f_X(x) =
      \begin{cases}
         &12(x-0.5)^2(1.5-x), &0.5 \leq x \leq 1.5\\
         &0, &\text{ Otherwise}
      \end{cases}

   1. Identify the support of :math:`X`, sketch :math:`f_X(x)`, then verify that
      it is a legitimate PDF.
      
      .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/potato-chip-pdf.png
         :alt: Sketch of the pdf of max diameter of potato chips
         :align: center
         :width: 60%

         Sketch of the PDF of maximum diamters of potato chips

      :math:`\text{supp}(X) = [0.5, 1.5]`. :math:`f_X(x)` takes a non-negative value
      at any :math:`x \in \mathbb{R}` as evident from the sketch. We now
      verify that the integral of the PDF from :math:`-\infty` to :math:`\infty`
      equals 1:
      
      .. math:: 
         \int_{-\infty}^\infty f_X(x)dx 
         &= \int_{-\infty}^{0.5} f_X(x)dx + \int_{0.5}^{1.5}f_X(x)dx + \int_{1.5}^{\infty} f_X(x)dx\\
         &= 0 + \int_{0.5}^{1.5} 12(x-0.5)^2(1.5-x) dx + 0\\                            

      Above, we first split the beginning intetral to a sum of integrals over three
      intervals. This step makes it evident that the integrals below :math:`0.5` and
      above :math:`1.5` are not just arbitrarily omitted from computationâ€”they simply 
      contribute zero area to the integral. Continuting,

      .. math:: 
         &\int_{0.5}^{1.5} 12(x-0.5)^2(1.5-x) dx\\
         &= \int_{0.5}^{1.5} (-12x^3 + 30x^2-21x + 4.5)dx\\
         &=\frac{-12x^4}{4} + \frac{30x^3}{3} - \frac{21x^2}{2} + 4.5x \Bigg\rvert_{0.5}^{1.5}\\
         &=-3x^4 + 10x^3 - 10.5x^2 + 4.5x \Bigg\rvert_{0.5}^{1.5}\\
         &= 1.6875-0.6875 = 1.

      Therefore, :math:`f_X` satisfies both requirements for a valid PDF.

   2. For a quality control procedure, managers of the factory has collected all the potato chips whose
      maximum diameter is **smaller than 1"**. What is the probability that
      a randomly selected potato chip in this pool has a maximum diameter **greater than 0.8"**?

      The first task is to write the goal of the problem in correct probability statement.
      Since we have the information that the chips will always have a maximum diameter less than 1,
      
      .. math::
        P(X > 0.8 | X < 1) = \frac{P(\{X > 0.8\} \cap \{X < 1 \})}{P(X < 1)}.

      The diameter can be greater than 0.8 **and** less than 1 only if it is between
      the two values. We simplify the numerator accordingly: 

      .. math::
        P(X < 0.8 | X < 1) = \frac{P(0.8 <X < 1)}{P(X < 1)}.

      Now integrate for each probability:

      * Numerator
      
         .. math:: 
            P(0.8 < X < 1) &= \int_{0.8}^{1} f_X(x)dx \\
            &=-3x^4 + 10x^3 - 10.5x^2 + 4.5x \Bigg\rvert_{0.8}^{1}\\
            &=0.2288
      
      * Denominator

        The denominator only has an upper boundary. This is equivalent to having 
        a lower boundary of negative infinity, which gives us
        
        .. math:: 
         P(X < 1) &= \int_{-\infty}^{1} f_X(x)dx = \int_{0.5}^1 f_X(x)dx\\
         

        The final step above is true because the integral of :math:`f_X(x)` in any region below
        :math:`0.5` results in 0. Continuing,

        .. math:: 
           P(X < 1) =-3x^4 + 10x^3 - 10.5x^2 + 4.5x \Bigg\rvert_{0.5}^{1} = 0.3125

      Finally, 
      
      .. math::
         P(X < 0.8 | X < 1) = \frac{0.2288}{0.3125} = 0.73216


Important Types of Problems invloving PDFs
-------------------------------------------------------

A. Handling Complex Distributions
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Real-world continuous distributions aren't always described by simple 
functions. Common examples of more complex distributions are:

.. _pdf_examples:
.. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/chapter6/examples-valid-pdf.png
   :alt: Example of a piecewise-defined probability density function
   :align: center
   :width: 80%

   *A legitimate PDF can have many different functional forms*

* PDFs with one or both ends of the support extending to infinity (left two on :numref:`pdf_examples`)
  
  - PDFs of this type usually includes an exponential function. Make sure to review 
    integration of simple exponential functions.

* Piecewise-defined PDFs taking different functional forms over 
  different regions of the support (right on :numref:`pdf_examples`)
  
  When dealing with piecewise PDFs,

   1. **Identify all boundaries** where the function forms change.
   2. **Break down** any integrals spanning multiple regions into
      a sum of integals, each spanning only a single region.

.. admonition:: ExampleðŸ’¡: A Piecewise-defined PDF
   :class: note

   A professor of STAT 1234 drops by a coffee shop on campus 
   before or after his  50-minute lecture. Suppose the time 
   he enters the coffee shop has a PDF shown below.
   The time :math:`x` is expressed in minutes relative to the start time of his lecture.

   .. math::
      f_X(x) = 
      \begin{cases}
      &\frac{7}{500}(x+10), &-10 \leq x \leq 0\\
      &\frac{3}{50}, &50 \leq x \leq 55\\
      &0, &\text{ Otherwise}
      \end{cases}

   What is the probability that the professor enters the cofee shop
   within 3 minutes from his lecture?

   We are looking for 

   .. math:: P(-3 \leq X \leq 53) = \int_{-3}^{53} f_X(x)dx.

   This interval spans three regions: 

   * part of the first non-trivial region where :math:`-3 \leq x\leq 0`, 
   * the middle region :math:`0 \leq x\leq 50` where the PDF is zero, then
   * part of the second non-trivial region where :math:`50\leq x\leq 53`. 
   
   We must break up the integral into three pieces accordingly:

   .. math:: P(-3 \leq X \leq 53) &= \int_{-3}^{0} f_X(x)dx + \int_{0}^{50} f_X(x)dx + \int_{50}^{53} f_X(x)dx\\
      &=\int_{-3}^{0} \frac{7}{500}(x+10) dx + \int_{0}^{50} 0 dx + \int_{50}^{53} \frac{3}{50}dx\\
      &=\frac{7}{500}\left(\frac{x^2}{2} + 10x\right)\Bigg\rvert_{-3}^0 + \frac{3x}{50}\Bigg\rvert_{50}^{53}\\
      &=0.357+0.18 = 0.537

B. Completing a Partially Known PDF
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Sometimes we encounter functions that have the right shape to be PDFs but don't integrate to 1.
We must fix this by finding an appropriate normalization constant.

.. admonition:: ExampleðŸ’¡: Finding the Normalization Constant
   :class: note 

   Suppose we have a function:
   
   .. math::
      g(x) = 
      \begin{cases}
      & x^2, & 0 \leq x \leq 2\\
      &0, &\text{ elsewhere}
      \end{cases}.

   Q: What is the constant :math:`k` which makes :math:`kg(x)` a valid PDF?

      We need to find :math:`k` such that :math:`\int_{-\infty}^{\infty} kg(x) \, dx = 1`.

      First, we compute the integral of :math:`g(x)`.

      .. math::
         \int_{-\infty}^{\infty}g(x)dx = \int_{0}^{2} x^2 \, dx = \left[\frac{x^3}{3}\right]_0^2 = \frac{8}{3}.

      To make the total area equal 1, we need

      .. math::
         k \frac{8}{3} = 1 \implies k = \frac{3}{8}.

      Therefore, the valid PDF is:

      .. math::
         f_X(x) = \begin{cases}
         \frac{3}{8}x^2 & \text{for } 0 \leq x \leq 2 \\
         0 & \text{elsewhere}
         \end{cases}.

Bringing It All Together
----------------------------

The transition from discrete to continuous random variables represents a fundamental shift in thinking. 
Instead of asking "what's the probability of exactly this value?" we ask "what's the probability of 
values in this range?" This change enables us to model the rich variety of measurement data we encounter.

.. admonition:: Key Takeaways ðŸ“
   :class: important

   1. **Continuous random variables** model measurable quantities that can take any value within an interval, 
      unlike discrete variables that count distinct outcomes.
   
   2. **Probability density functions (PDFs)** describe how probability is distributed across the continuous 
      range, with higher values indicating more likely regions.

   3. **Probabilities are areas** under the PDF, computed using integration: 
      :math:`P(a < X < b) = \int_a^b f_X(x) \, dx`.

   4. **Individual points have probability zero** for continuous variablesâ€”only intervals have positive probability.
   

   5. **Valid PDFs must be non-negative everywhere** and **integrate to 1** over their entire support.

In the next section, we'll explore how to calculate expected values and variances for continuous 
random variables, extending the concepts we mastered in the discrete case.

Exercises

These exercises develop your skills in working with probability density functions, computing probabilities through integration, and verifying PDF validity.

.. admonition:: Exercise 1: Validating and Computing Probabilities with a Basic PDF
   :class: note

   A mechanical engineer models the deviation :math:`X` (in millimeters) of a machined component from its target dimension. Based on production data, the PDF is:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{3}{4}(1 - x^2), & -1 \leq x \leq 1\\
      0, & \text{elsewhere}
      \end{cases}

   a. Verify that :math:`f_X(x)` is a valid PDF by checking both required properties.

   b. Sketch the PDF. What shape does it have? Is it symmetric?

   c. Find :math:`P(-0.5 \leq X \leq 0.5)`, the probability that the deviation is within half a millimeter of target.

   d. Find :math:`P(X > 0)`. Does your answer make sense given the symmetry?

   e. Find :math:`P(X = 0.3)`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF validity**

      **Property 1: Non-negativity**
      
      We need :math:`f_X(x) \geq 0` for all :math:`x`. On the support :math:`[-1, 1]`:
      
      - At :math:`x = 0`: :math:`f_X(0) = \frac{3}{4}(1-0) = \frac{3}{4} > 0`
      - At :math:`x = \pm 1`: :math:`f_X(\pm 1) = \frac{3}{4}(1-1) = 0 \geq 0`
      - For :math:`-1 < x < 1`: since :math:`|x| < 1`, we have :math:`x^2 < 1`, so :math:`1 - x^2 > 0`
      
      Therefore, :math:`f_X(x) \geq 0` everywhere. âœ“

      **Property 2: Total area equals 1**

      .. math::

         \int_{-\infty}^{\infty} f_X(x) \, dx &= \int_{-1}^{1} \frac{3}{4}(1 - x^2) \, dx\\
         &= \frac{3}{4} \left[ x - \frac{x^3}{3} \right]_{-1}^{1}\\
         &= \frac{3}{4} \left[ \left(1 - \frac{1}{3}\right) - \left(-1 + \frac{1}{3}\right) \right]\\
         &= \frac{3}{4} \left[ \frac{2}{3} + \frac{2}{3} \right] = \frac{3}{4} \cdot \frac{4}{3} = 1 \text{ âœ“}

      Both properties are satisfied, so :math:`f_X(x)` is a valid PDF.

      **Part (b): Sketch and shape**

      The PDF is a downward-opening parabola with:
      
      - Maximum at :math:`x = 0` where :math:`f_X(0) = \frac{3}{4}`
      - Zeros at :math:`x = \pm 1`
      
      The function is **symmetric** about :math:`x = 0` since :math:`f_X(-x) = \frac{3}{4}(1 - (-x)^2) = \frac{3}{4}(1 - x^2) = f_X(x)`.

      .. image:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-1/fig1_parabolic_pdf.png
         :alt: Parabolic PDF with shaded region showing P(-0.5 â‰¤ X â‰¤ 0.5)
         :align: center
         :width: 550px

      **Part (c): P(-0.5 â‰¤ X â‰¤ 0.5)**

      .. math::

         P(-0.5 \leq X \leq 0.5) &= \int_{-0.5}^{0.5} \frac{3}{4}(1 - x^2) \, dx\\
         &= \frac{3}{4} \left[ x - \frac{x^3}{3} \right]_{-0.5}^{0.5}\\
         &= \frac{3}{4} \left[ \left(0.5 - \frac{0.125}{3}\right) - \left(-0.5 + \frac{0.125}{3}\right) \right]\\
         &= \frac{3}{4} \left[ 1 - \frac{0.25}{3} \right] = \frac{3}{4} \cdot \frac{11}{12} = \frac{11}{16} = 0.6875

      About **68.75%** of components have deviations within Â±0.5 mm.

      **Part (d): P(X > 0)**

      By symmetry of :math:`f_X(x)` about :math:`x = 0`:

      .. math::

         P(X > 0) = P(X < 0) = \frac{1}{2} = 0.5

      This makes senseâ€”the symmetric PDF assigns equal probability to positive and negative deviations.

      **Part (e): P(X = 0.3)**

      For any continuous random variable:

      .. math::

         P(X = 0.3) = \int_{0.3}^{0.3} f_X(x) \, dx = 0

      **Any single point has probability zero** for continuous random variables, even though :math:`x = 0.3` is in the support and has positive density.

.. admonition:: Exercise 2: PDF with Non-Zero Starting Point (Support Not Starting at 0)
   :class: note

   A biomedical engineer studies the response time :math:`X` (in seconds) of a nerve impulse. The PDF is:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{1}{8}(x - 2), & 2 \leq x \leq 6\\
      0, & \text{elsewhere}
      \end{cases}

   Note that the support begins at :math:`x = 2`, not :math:`x = 0`.

   a. Verify this is a valid PDF.

   b. Find :math:`P(3 \leq X \leq 5)` by direct integration.

   c. Find :math:`P(X > 4)`.

   d. **Common Mistake Alert**: A student incorrectly computes :math:`P(X \leq 4)` by integrating :math:`\int_0^4 f_X(x) \, dx`. Explain their error and compute the correct value.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF validity**

      **Non-negativity**: On :math:`[2, 6]`, since :math:`x \geq 2`, we have :math:`x - 2 \geq 0`, so :math:`f_X(x) = \frac{1}{8}(x-2) \geq 0`. âœ“

      **Total area equals 1**:

      .. math::

         \int_{-\infty}^{\infty} f_X(x) \, dx &= \int_{2}^{6} \frac{1}{8}(x - 2) \, dx\\
         &= \frac{1}{8} \cdot \frac{(x-2)^2}{2} \Bigg|_{2}^{6}\\
         &= \frac{1}{16} \left[ (6-2)^2 - (2-2)^2 \right]\\
         &= \frac{1}{16} \cdot 16 = 1 \text{ âœ“}

      .. image:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-1/fig2a_nonzero_start_pdf.png
         :alt: PDF with support starting at x=2
         :align: center
         :width: 550px

      **Part (b): P(3 â‰¤ X â‰¤ 5) by direct integration**

      .. math::

         P(3 \leq X \leq 5) &= \int_{3}^{5} \frac{1}{8}(x - 2) \, dx\\
         &= \frac{1}{8} \cdot \frac{(x-2)^2}{2} \Bigg|_{3}^{5}\\
         &= \frac{1}{16} \left[ (5-2)^2 - (3-2)^2 \right]\\
         &= \frac{1}{16} \left[ 9 - 1 \right] = \frac{8}{16} = 0.5

      **Part (c): P(X > 4)**

      .. math::

         P(X > 4) &= \int_{4}^{6} \frac{1}{8}(x - 2) \, dx\\
         &= \frac{1}{16} \left[ (6-2)^2 - (4-2)^2 \right]\\
         &= \frac{1}{16} \left[ 16 - 4 \right] = \frac{12}{16} = 0.75

      **Part (d): Common Mistake Explanation**

      The student's error: They integrated from 0, but the PDF equals 0 for :math:`x < 2`. The correct approach:

      .. math::

         P(X \leq 4) = \int_{-\infty}^{4} f_X(t) \, dt = \underbrace{\int_{-\infty}^{2} f_X(t) \, dt}_{= 0} + \int_{2}^{4} \frac{1}{8}(t-2) \, dt = \frac{(4-2)^2}{16} = \frac{4}{16} = 0.25

      **Key insight**: When computing probabilities, only the region where :math:`f_X(x) > 0` contributes. When the support doesn't start at 0, be careful to integrate from the actual start of the support.

.. admonition:: Exercise 3: Piecewise PDF with Multiple Regions
   :class: note

   An aerospace engineer models wind gust intensity :math:`X` (in m/s deviation from average) with the following PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      \frac{1}{9}(x + 3), & -3 \leq x \leq 0\\
      \frac{1}{9}(3 - x), & 0 < x \leq 3\\
      0, & \text{elsewhere}
      \end{cases}

   a. Verify this is a valid PDF. What is the geometric shape?

   b. Use geometry (not calculus) to find :math:`P(-1 \leq X \leq 1)`.

   c. Find :math:`P(X > 1)` using integration.

   d. Find the probability that the wind gust intensity exceeds 1 m/s given that it is positive: :math:`P(X > 1 | X > 0)`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Verify PDF and identify shape**

      The PDF forms a **triangle** with:
      
      - Vertex at :math:`(0, \frac{3}{9}) = (0, \frac{1}{3})`
      - Base from :math:`x = -3` to :math:`x = 3` (length = 6)
      
      **Non-negativity**: Both pieces are non-negative on their respective domains. âœ“

      **Total area = 1** (using triangle area formula):

      .. math::

         \text{Area} = \frac{1}{2} \times \text{base} \times \text{height} = \frac{1}{2} \times 6 \times \frac{1}{3} = 1 \text{ âœ“}

      .. image:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-1/fig3a_triangular_pdf.png
         :alt: Triangular piecewise PDF symmetric about x=0
         :align: center
         :width: 550px

      **Part (b): Geometric probability calculation**

      The region :math:`-1 \leq x \leq 1` forms a trapezoid inside the triangle.

      At :math:`x = -1`: :math:`f_X(-1) = \frac{1}{9}(-1+3) = \frac{2}{9}`
      
      At :math:`x = 0`: :math:`f_X(0) = \frac{1}{3}`
      
      At :math:`x = 1`: :math:`f_X(1) = \frac{1}{9}(3-1) = \frac{2}{9}`

      **Alternative geometric approach**: 
      
      Total triangle area = 1. The triangles outside :math:`[-1, 1]`:
      
      - Left triangle (:math:`-3 \leq x \leq -1`): base = 2, height = :math:`\frac{2}{9}`, area = :math:`\frac{1}{2} \times 2 \times \frac{2}{9} = \frac{2}{9}`
      - Right triangle (:math:`1 \leq x \leq 3`): base = 2, height = :math:`\frac{2}{9}`, area = :math:`\frac{2}{9}`

      .. math::

         P(-1 \leq X \leq 1) = 1 - \frac{2}{9} - \frac{2}{9} = 1 - \frac{4}{9} = \frac{5}{9} \approx 0.556

      **Part (c): P(X > 1) using integration**

      .. math::

         P(X > 1) &= \int_{1}^{3} \frac{1}{9}(3 - x) \, dx\\
         &= \frac{1}{9}\left[3x - \frac{x^2}{2}\right]_{1}^{3}\\
         &= \frac{1}{9}\left[\left(9 - \frac{9}{2}\right) - \left(3 - \frac{1}{2}\right)\right]\\
         &= \frac{1}{9}\left[\frac{9}{2} - \frac{5}{2}\right] = \frac{1}{9} \cdot 2 = \frac{2}{9}

      **Part (d): Conditional probability P(X > 1 | X > 0)**

      .. math::

         P(X > 1 | X > 0) = \frac{P(X > 1 \cap X > 0)}{P(X > 0)} = \frac{P(X > 1)}{P(X > 0)}

      By symmetry: :math:`P(X > 0) = 0.5`

      From part (c): :math:`P(X > 1) = \frac{2}{9}`

      .. math::

         P(X > 1 | X > 0) = \frac{2/9}{1/2} = \frac{4}{9} \approx 0.444

.. admonition:: Exercise 4: Finding the Normalizing Constant
   :class: note

   A chemical engineer proposes the following function to model reaction time :math:`X` (in minutes):

   .. math::

      g(x) = 
      \begin{cases}
      c(4x - x^2), & 0 \leq x \leq 4\\
      0, & \text{elsewhere}
      \end{cases}

   a. Find the value of :math:`c` that makes :math:`g(x)` a valid PDF.

   b. With this value of :math:`c`, find :math:`P(1 \leq X \leq 3)`.

   c. Sketch the PDF. Where is the mode (peak) of this distribution?

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Find normalizing constant c**

      For a valid PDF, :math:`\int_{-\infty}^{\infty} g(x) \, dx = 1`:

      .. math::

         \int_{0}^{4} c(4x - x^2) \, dx = 1

      .. math::

         c \left[ 2x^2 - \frac{x^3}{3} \right]_{0}^{4} = 1

      .. math::

         c \left[ 2(16) - \frac{64}{3} \right] = c \left[ 32 - \frac{64}{3} \right] = c \cdot \frac{32}{3} = 1

      .. math::

         c = \frac{3}{32}

      **Part (b): P(1 â‰¤ X â‰¤ 3)**

      .. math::

         P(1 \leq X \leq 3) &= \int_{1}^{3} \frac{3}{32}(4x - x^2) \, dx\\

      Solving: :math:`x = 2`

      The **mode is at x = 2 minutes**, where :math:`f_X(2) = \frac{3}{32}(8 - 4) = \frac{3}{8}`.

.. admonition:: Exercise 5: Piecewise PDF with Gap in Support
   :class: note

   A data scientist models server response categories with the following PDF for response time :math:`X` (in milliseconds):

   .. math::

      f_X(x) = 
      \begin{cases}

   .. dropdown:: Solution
      :class-container: sd-border-success

.. admonition:: Exercise 6: Exponential-Type Function
   :class: note

   A reliability engineer models component lifetime :math:`X` (in years) with PDF:

   .. math::

      f_X(x) = 
      \begin{cases}
      ce^{-x/2}, & x \geq 0\\
      0, & x < 0
      \end{cases}

   a. Find the constant :math:`c` that makes this a valid PDF.

   b. Find :math:`P(X > 3)`, the probability a component lasts more than 3 years.

   c. Find :math:`P(1 < X < 4)`.

   d. If a component has already lasted 2 years, what is :math:`P(X > 5 | X > 2)`?

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Find c**

      .. math::

         \int_{0}^{\infty} ce^{-x/2} \, dx = c \cdot \left[-2e^{-x/2}\right]_{0}^{\infty} = c \cdot (0 - (-2)) = 2c = 1

      Therefore, :math:`c = \frac{1}{2}`.

      The PDF is :math:`f_X(x) = \frac{1}{2}e^{-x/2}` for :math:`x \geq 0`.

      .. image:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-1/fig7a_exponential_pdf.png
         :alt: Exponential PDF
         :align: center
         :width: 550px

      **Part (b): P(X > 3)**

      .. math::

         P(X > 3) = \int_{3}^{\infty} \frac{1}{2}e^{-x/2} \, dx = \left[-e^{-x/2}\right]_{3}^{\infty} = 0 - (-e^{-3/2}) = e^{-1.5} \approx 0.223

      **Part (c): P(1 < X < 4)**

      .. math::

         P(1 < X < 4) = \int_{1}^{4} \frac{1}{2}e^{-x/2} \, dx = \left[-e^{-x/2}\right]_{1}^{4} = -e^{-2} - (-e^{-0.5}) = e^{-0.5} - e^{-2} \approx 0.471

      **Part (d): Conditional probability (Memoryless property)**

      This is an exponential distribution with rate :math:`\lambda = \frac{1}{2}`. By the **memoryless property**:

      .. math::

         P(X > 5 | X > 2) = P(X > 3) = e^{-3/2} \approx 0.223

      The probability of surviving an additional 3 years is the same regardless of how long the component has already lastedâ€”this is the defining characteristic of exponential distributions.

.. admonition:: Exercise 8: Finding Normalizing Constants â€” Exponential Functions
   :class: note

   For each of the following functions, find the constant :math:`c` that makes it a valid PDF.

   a. :math:`g(x) = ce^{-3x}` for :math:`x \geq 0`, and 0 elsewhere.

   b. :math:`g(x) = ce^{-x/5}` for :math:`x \geq 0`, and 0 elsewhere.

   c. :math:`g(x) = cxe^{-x}` for :math:`x \geq 0`, and 0 elsewhere. (Hint: Use integration by parts)

   d. :math:`g(x) = ce^{-(x-2)}` for :math:`x \geq 2`, and 0 elsewhere. (Note: Support starts at 2, not 0)

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): g(x) = ce^{-3x} for x â‰¥ 0**

      .. math::

         \int_{0}^{\infty} ce^{-3x} \, dx = c \cdot \left[ -\frac{1}{3}e^{-3x} \right]_{0}^{\infty} = c \cdot \left( 0 - \left(-\frac{1}{3}\right) \right) = \frac{c}{3} = 1

      Therefore, :math:`c = 3`.

      The PDF is :math:`f_X(x) = 3e^{-3x}` for :math:`x \geq 0` (Exponential with rate :math:`\lambda = 3`).

      **Part (b): g(x) = ce^{-x/5} for x â‰¥ 0**

      .. math::

         \int_{0}^{\infty} ce^{-x/5} \, dx = c \cdot \left[ -5e^{-x/5} \right]_{0}^{\infty} = c \cdot (0 - (-5)) = 5c = 1

      Therefore, :math:`c = \frac{1}{5}`.

      The PDF is :math:`f_X(x) = \frac{1}{5}e^{-x/5}` for :math:`x \geq 0` (Exponential with rate :math:`\lambda = \frac{1}{5}`, or mean :math:`\mu = 5`).

      **Part (c): g(x) = cxe^{-x} for x â‰¥ 0**

      Use integration by parts with :math:`u = x`, :math:`dv = e^{-x}dx`:

      Then :math:`du = dx`, :math:`v = -e^{-x}`.

      .. math::

         \int_{0}^{\infty} cxe^{-x} \, dx &= c \left[ -xe^{-x} \Big|_{0}^{\infty} + \int_{0}^{\infty} e^{-x} \, dx \right]\\
         &= c \left[ 0 + \left( -e^{-x} \Big|_{0}^{\infty} \right) \right]\\
         &= c \left[ 0 - (-1) \right] = c = 1

      Therefore, :math:`c = 1`.

      The PDF is :math:`f_X(x) = xe^{-x}` for :math:`x \geq 0`. This is a **Gamma distribution** with shape :math:`\alpha = 2` and rate :math:`\beta = 1`.

      **Part (d): g(x) = ce^{-(x-2)} for x â‰¥ 2**

      The support starts at :math:`x = 2`, so we integrate from 2:

      .. math::

         \int_{2}^{\infty} ce^{-(x-2)} \, dx = c \cdot \left[ -e^{-(x-2)} \right]_{2}^{\infty} = c \cdot (0 - (-1)) = c = 1

      Therefore, :math:`c = 1`.

      The PDF is :math:`f_X(x) = e^{-(x-2)}` for :math:`x \geq 2`. This is a **shifted exponential** distribution.

.. admonition:: Exercise 9: Piecewise Triangular PDF
   :class: note

   A computer scientist models network packet sizes :math:`X` (in kilobytes) with the following proposed density:

   .. math::

      g(x) = 
      \begin{cases}
      cx, & 0 \leq x \leq 2\\
      c(4 - x), & 2 < x \leq 4\\
      0, & \text{elsewhere}
      \end{cases}

   a. Find the constant :math:`c` that makes this a valid PDF.

   b. Sketch the PDF. What shape is it?

   c. Find :math:`P(1 < X < 3)`.

   d. Find :math:`P(X > 3)`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Part (a): Find normalizing constant c**

      The PDF must integrate to 1 over the entire support:

      .. math::

         \int_{0}^{2} cx \, dx + \int_{2}^{4} c(4-x) \, dx = 1

      **First integral**:

      .. math::

         \int_{0}^{2} cx \, dx = c \cdot \frac{x^2}{2} \Bigg|_{0}^{2} = c \cdot 2 = 2c

      **Second integral**:

      .. math::

         \int_{2}^{4} c(4-x) \, dx = c \left[ 4x - \frac{x^2}{2} \right]_{2}^{4} = c \left[ (16 - 8) - (8 - 2) \right] = c \cdot 2 = 2c

      **Total**: :math:`2c + 2c = 4c = 1`, so :math:`c = \frac{1}{4}`.

      **Part (b): Sketch and shape**

      The PDF forms a **triangle** with:
      
      - Base from :math:`x = 0` to :math:`x = 4` (length = 4)
      - Peak at :math:`x = 2` where :math:`f_X(2) = \frac{1}{4} \cdot 2 = \frac{1}{2}`
      
      **Verification using triangle area**: :math:`\frac{1}{2} \times 4 \times \frac{1}{2} = 1` âœ“

      .. image:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/Exercises/ch6-1/fig10a_piecewise_triangular_pdf.png
         :alt: Triangular piecewise PDF with c = 1/4
         :align: center
         :width: 550px

      **Part (c): P(1 < X < 3)**

      This spans both pieces of the PDF:

      .. math::

         P(1 < X < 3) &= \int_{1}^{2} \frac{x}{4} \, dx + \int_{2}^{3} \frac{4-x}{4} \, dx\\
         &= \frac{1}{4}\left[\frac{x^2}{2}\right]_{1}^{2} + \frac{1}{4}\left[4x - \frac{x^2}{2}\right]_{2}^{3}\\
         &= \frac{1}{4}\left[2 - \frac{1}{2}\right] + \frac{1}{4}\left[\left(12 - \frac{9}{2}\right) - \left(8 - 2\right)\right]\\
         &= \frac{1}{4} \cdot \frac{3}{2} + \frac{1}{4} \cdot \frac{3}{2} = \frac{3}{8} + \frac{3}{8} = \frac{3}{4}

      **Part (d): P(X > 3)**

      .. math::

         P(X > 3) = \int_{3}^{4} \frac{4-x}{4} \, dx = \frac{1}{4}\left[4x - \frac{x^2}{2}\right]_{3}^{4} = \frac{1}{4}\left[8 - \frac{15}{2}\right] = \frac{1}{4} \cdot \frac{1}{2} = \frac{1}{8}
