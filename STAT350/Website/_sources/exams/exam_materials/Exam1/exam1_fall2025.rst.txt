.. admonition:: Question 1.1  (2 pts)
   :class: note

   Let :math:`X` and :math:`Y` be two discrete random variables with supports
   :math:`x \in \{0, 1, 2\}` and :math:`y \in \{1, 2, 3\}`.

   **T or F:** If :math:`P(X=1,\; Y=1) = P(X=1)\cdot P(Y=1)`, then this implies that
   :math:`X` and :math:`Y` are independent random variables.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      Independence requires that the product rule holds for **every** pair of values in the
      joint support:

      .. math::

         P(X = x,\; Y = y) = P(X = x)\cdot P(Y = y)
         \quad \text{for all } x \in \{0,1,2\},\; y \in \{1,2,3\}.

      The condition given — :math:`P(X=1, Y=1) = P(X=1)\cdot P(Y=1)` — verifies the product
      rule for only **one** of the :math:`3 \times 3 = 9` required pairs. All nine must hold
      simultaneously for independence to be established. Satisfying just one tells us nothing
      about the remaining eight pairs.

      The statement is **FALSE**.

.. admonition:: Question 1.2  (2 pts)
   :class: note

   For three events :math:`A`, :math:`B`, and :math:`C` from the same sample space
   :math:`\Omega`, it is known that :math:`C \neq \emptyset` and :math:`P(A \cap B) > 0`.

   **T or F:** If :math:`P(A \cap B \cap C) = 0`, it must follow that
   :math:`P(A \cap C) = P(B \cap C) = 0`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      Just because no outcome belongs to all three sets simultaneously does not mean that
      :math:`A` and :math:`C`, or :math:`B` and :math:`C`, cannot overlap pairwise.

      **Counterexample:** Let :math:`\Omega = \{1, 2, 3, 4\}` with each outcome equally
      likely (probability 1/4). Define:

      .. math::

         A = \{1, 2\}, \qquad B = \{2, 3\}, \qquad C = \{1, 3\}.

      Check the conditions:

      .. math::

         P(A \cap B) = P(\{2\}) = \tfrac{1}{4} > 0. \quad C \neq \emptyset. \checkmark

      .. math::

         P(A \cap B \cap C) = P(\emptyset) = 0. \checkmark

      But:

      .. math::

         P(A \cap C) = P(\{1\}) = \tfrac{1}{4} \neq 0.

      So :math:`P(A \cap B \cap C) = 0` does **not** force :math:`P(A \cap C) = 0`.
      The statement is **FALSE**.

.. admonition:: Question 1.3  (2 pts)
   :class: note

   Let :math:`X` be a continuous random variable with a PDF :math:`f_X(x)` defined over the
   symmetric interval :math:`[-c, c]` for some constant :math:`c > 0`.

   **T or F:** If the PDF is an even function (meaning :math:`f_X(-x) = f_X(x)` for all
   :math:`x`), then this implies that the 50th percentile (median) of the distribution must
   be 0.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      If :math:`f_X` is an even function on :math:`[-c, c]`, the distribution is symmetric about
      zero. Evaluate the CDF at :math:`x = 0`:

      .. math::

         F_X(0) = \int_{-c}^{0} f_X(x)\,dx.

      By the even-function property :math:`f_X(-x) = f_X(x)`, the substitution :math:`u = -x`
      gives:

      .. math::

         \int_{-c}^{0} f_X(x)\,dx = \int_{0}^{c} f_X(u)\,du.

      Since the total area equals 1:

      .. math::

         F_X(0) = \int_{-c}^{0} f_X(x)\,dx = \frac{1}{2}.

      Because :math:`F_X(0) = 0.5`, the median is 0. The statement is **TRUE**.

.. admonition:: Question 1.4  (2 pts)
   :class: note

   A censor reports a raw error measurement :math:`X` in millivolts. Historical data show a
   bias in these measurements with :math:`E[X] = -2` (millivolts) and
   :math:`\text{SD}(X) = 5` (millivolts).

   **T or F:** For the transformed score :math:`Y = 5X + 35`, it follows that
   :math:`E[Y] = \text{SD}(Y)`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      Apply the linear transformation rules.

      **Expected value:**

      .. math::

         E[Y] = E[5X + 35] = 5\,E[X] + 35 = 5(-2) + 35 = -10 + 35 = 25.

      **Standard deviation:**

      .. math::

         \text{SD}(Y) = |5|\,\text{SD}(X) = 5 \times 5 = 25.

      Since :math:`E[Y] = 25 = \text{SD}(Y)`, the statement is **TRUE**.

.. admonition:: Question 1.5  (2 pts)
   :class: note

   In Normal distribution word problems, we distinguish "forward" and "backward" problems.

   **T or F:** "Backward" problems solve for an :math:`x`-value given a probability, while
   "forward" problems solve for a probability given an :math:`x`-value.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      This is the standard terminology used in Normal distribution calculations:

      - A **forward problem** starts with an :math:`x`-value (or range of :math:`x`-values)
        and asks for a probability. The workflow is: standardize to obtain a :math:`z`-score,
        then look up the probability in the :math:`z`-table.

      - A **backward problem** starts with a probability (a percentile) and asks for the
        corresponding :math:`x`-value. The workflow is: find the :math:`z`-score from the
        :math:`z`-table, then reverse-transform:
        :math:`x = \mu + z\sigma`.

      The statement is **TRUE**.

.. admonition:: Question 1.6  (2 pts)
   :class: note

   In a class of 60 students, the instructor randomly selects 20 homework submissions
   **without replacement** to audit for possible AI policy violations. Each audited paper is
   labeled *probable violation* or *no obvious violation*. Let :math:`X` be the number of
   audited papers with probable violations.

   **T or F:** Then :math:`X` is a Binomial random variable because the process satisfies
   the BINS conditions.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      Recall the four **BINS** conditions required for a Binomial distribution:

      - **B** — Binary outcomes (probable violation / no obvious violation ✓)
      - **I** — Independent trials
      - **N** — Fixed number of trials (n = 20 ✓)
      - **S** — Same probability of success on every trial

      Sampling **without replacement** from a finite population of 60 violates both the
      **I** and **S** conditions. After each paper is selected, the composition of the
      remaining pool changes, so the probability of drawing a violation changes from draw
      to draw, and the draws are not independent.

      Since the **I** (independent trials) and **S** (same probability) conditions are both
      violated, :math:`X` is **not** a Binomial random variable. The statement is **FALSE**.

.. admonition:: Question 2.1  (3 pts)
   :class: note

   A researcher randomly selected 100 graduate students and surveyed their daily expenses on
   eating out. The collected data are visualized in the boxplot below.

   .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/Exams/Exam1/FALL2025/image1.png
      :alt: A horizontal modified boxplot titled "Daily Expenses ($) - Uncorrected". The box spans approximately $3 to $15, with median near $7 and mean dot near $10. The lower whisker extends left to about $0. The upper whisker extends right to approximately $31. One explicit point (outlier) appears to the far right at approximately $33, representing the value 37.24.
      :align: center
      :width: 85%

   The researcher received an email from one of the participants, stating that they had
   misreported the amount, and the corrected amount should be **27.24** instead of **37.24**.
   After this correction, which of the **two** graphical components of the boxplot remain
   unchanged?

   - **(A)** Third Quartile, :math:`Q_3`
   - **(B)** Sample Mean, :math:`\bar{x}`
   - **(C)** Upper Whisker
   - **(D)** The number of explicit points
   - **(E)** The maximum value

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (A) and (C)**

      The key question is whether 27.24 remains an outlier after the correction.

      From the boxplot, approximate values are: :math:`Q_1 \approx 3`,
      :math:`Q_3 \approx 15`, giving :math:`IQR \approx 12`.

      .. math::

         \text{Upper inner fence} = Q_3 + 1.5 \times IQR \approx 15 + 18 = 33.

      Since :math:`37.24 > 33`, the original value was an outlier (explicit point). ✓

      After correction: :math:`27.24 < 33`, so **27.24 is no longer an outlier** — it falls
      within the upper fence and is absorbed into the regular data.

      Now evaluate each component:

      - **(A) Q₃**: :math:`Q_3` is determined by the middle 50% of the data. The corrected
        value 37.24 → 27.24 was an upper outlier, far above :math:`Q_3 \approx 15`. Moving
        it to 27.24 still leaves it above :math:`Q_3`, so :math:`Q_3` is **unchanged**. ✓

      - **(B) Sample Mean**: The mean decreases by :math:`(37.24 - 27.24)/100 = 0.10`.
        **Changes.** ✗

      - **(C) Upper Whisker**: The upper whisker endpoint is the largest observation that
        falls at or below the upper inner fence (the largest non-outlier). Originally 37.24
        was the sole outlier, so the whisker endpoint was the next-largest value (:math:`\approx 31`).
        After the correction, 27.24 is no longer an outlier, but :math:`27.24 < 31`, so the
        largest non-outlier is still :math:`\approx 31`. The upper whisker is **unchanged**. ✓

      - **(D) Number of explicit points**: Before: 1 outlier (37.24). After: 0 outliers (27.24
        is now within the fence). **Changes.** ✗

      - **(E) Maximum value**: The overall maximum was 37.24; it is now 27.24 (or whichever
        value was previously the upper whisker endpoint, now the new maximum). **Changes.** ✗

      The two unchanged components are **(A) Q₃** and **(C) Upper Whisker**.

.. admonition:: Question 2.2  (3 pts)
   :class: note

   At a large clinic, women's heights are well modeled by a Normal distribution with mean
   165 cm and standard deviation 7 cm. Identify the **false** statement from A–E.

   - **(A)** The mean, median, and mode are all 165 cm.
   - **(B)** The 25th and 75th percentiles are equidistant from 165 cm.
   - **(C)** Converting heights from centimeters to inches does not change anyone's z-score
     or percentile.
   - **(D)** About 68% of women have heights within 7 cm of 165 cm.
   - **(E)** Every woman's height lies within 3 standard deviations of 165 cm.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (E)**

      Evaluate each statement:

      **(A) TRUE.** For any symmetric, unimodal distribution — and the Normal distribution
      is both — the mean, median, and mode coincide at the center :math:`\mu = 165` cm.

      **(B) TRUE.** By symmetry of the Normal distribution about :math:`\mu = 165` cm,
      :math:`Q_1 = 165 - z_{0.75} \times 7` and :math:`Q_3 = 165 + z_{0.75} \times 7`.
      From the z-table, :math:`z_{0.75} = 0.67`, so both quartiles are exactly
      :math:`0.67 \times 7 = 4.69` cm from 165 cm — equidistant.

      **(C) TRUE.** A linear unit conversion :math:`h_{\text{in}} = h_{\text{cm}} / 2.54`
      scales both the observation and the mean by the same factor, leaving
      :math:`z = (h - \mu)/\sigma` unchanged.

      **(D) TRUE.** This is the 68% part of the empirical rule: approximately 68% of
      observations from a Normal distribution fall within :math:`\pm 1` standard deviation of
      the mean, i.e., within 7 cm of 165 cm.

      **(E) FALSE.** The Normal distribution has **infinite support** — it extends from
      :math:`-\infty` to :math:`+\infty`. While approximately 99.7% of women have heights
      within 3 standard deviations of 165 cm (i.e., between 144 cm and 186 cm), there is
      no finite bound beyond which the probability is exactly zero. It is theoretically
      possible (with very small probability) for an observation to fall more than 3 SDs from
      the mean. The statement is **FALSE**.

.. admonition:: Question 2.3  (3 pts)
   :class: note

   A clinical trial enrolls 10 women, each receiving one dose of a new drug. Based on a
   pre-measured genotype, 4 participants have a 30% chance of response (type :math:`G_1`)
   and 6 have a 70% chance (type :math:`G_2`). Responses are independent across
   participants, and each participant's outcome is either response or no response. Let
   :math:`X` be the number who respond. Identify the **correct** statement from A–E.

   - **(A)** The number of trials is not a fixed constant.
   - **(B)** The trials are dependent.
   - **(C)** There are more than two possible outcomes on each trial.
   - **(D)** The probability of success is not the same for all trials.
   - **(E)** The trials are conducted without replacement from a small finite population.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      This question tests the **BINS conditions** for a Binomial distribution. Only one of
      the four conditions fails here, and the correct answer identifies which.

      **(A) FALSE.** The number of trials is fixed: :math:`n = 10` women, each tested exactly
      once.

      **(B) FALSE.** The problem explicitly states that responses are independent across
      participants.

      **(C) FALSE.** Each outcome is binary: *response* or *no response*. There are exactly
      two possible outcomes per trial.

      **(D) TRUE.** The 4 type-:math:`G_1` participants each have probability 0.30 of
      responding, while the 6 type-:math:`G_2` participants each have probability 0.70.
      Since :math:`0.30 \neq 0.70`, the **probability of success varies across trials**,
      violating the **S** (Same probability) condition of BINS. :math:`X` is therefore
      **not** a Binomial random variable.

      **(E) FALSE.** The participants are distinct individuals, not drawn from a shared
      pool without replacement; each is tested exactly once.

.. admonition:: Question 2.4  (3 pts)
   :class: note

   A new type of biodegradable plastic is developed, and its degradation time :math:`X`
   (in years) is modeled by the probability density function

   .. math::

      f_X(x) = \begin{cases} k\,x^2 & 0 \leq x \leq 3 \\ 0 & \text{otherwise} \end{cases}

   What is the expected lifetime of this plastic?

   - **(A)** 1 year
   - **(B)** 1.5 years
   - **(C)** 2.25 years
   - **(D)** 3 years
   - **(E)** 3.25 years
   - **(F)** 9 years

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (C)**

      **Step 1: Find k.**

      The PDF must integrate to 1:

      .. math::

         \int_0^3 k\,x^2\,dx = k\!\left[\frac{x^3}{3}\right]_0^3
         = k \cdot \frac{27}{3} = 9k = 1
         \implies k = \frac{1}{9}.

      **Step 2: Compute** :math:`E[X]`.

      .. math::

         E[X] = \int_0^3 x \cdot \frac{1}{9}\,x^2\,dx
              = \frac{1}{9}\int_0^3 x^3\,dx
              = \frac{1}{9}\!\left[\frac{x^4}{4}\right]_0^3
              = \frac{1}{9} \cdot \frac{81}{4}
              = \frac{81}{36} = \boxed{2.25 \text{ years}}.

      The answer is **(C)**.

.. admonition:: Question 2.5  (3 pts)
   :class: note

   Let :math:`X \sim \text{Binomial}(n=2,\; p)` with unknown :math:`p \in (0,1)`. Define a
   second random variable :math:`Y` conditionally on :math:`X` as follows:

   - :math:`P(Y = 0 \mid X = 0) = 1`
   - :math:`P(Y = 1 \mid X = 1) = P(Y = 2 \mid X = 1) = \dfrac{1}{2}`
   - :math:`P(Y = y \mid X = 2) \sim \text{Poisson}(\lambda)`

   Which expression equals :math:`P(Y = 0)`?

   - **(A)** :math:`(1-p)^2 + p^2 \cdot e^{-\lambda}`
   - **(B)** :math:`(1-p)^2 \cdot e^{-\lambda} + p^2`
   - **(C)** :math:`(1-p)^2 + 2p(1-p) + p^2 \cdot e^{-\lambda}`
   - **(D)** :math:`(1-p) \cdot p \cdot e^{-\lambda}`

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (A)**

      Apply the **Law of Total Probability**, conditioning on :math:`X`:

      .. math::

         P(Y=0) = \sum_{x=0}^{2} P(Y=0 \mid X=x)\,P(X=x).

      **Marginal probabilities of X** (Binomial with n=2):

      .. math::

         P(X=0) = (1-p)^2, \qquad
         P(X=1) = 2p(1-p), \qquad
         P(X=2) = p^2.

      **Conditional probabilities of Y=0:**

      - :math:`P(Y=0 \mid X=0) = 1` (given directly).
      - :math:`P(Y=0 \mid X=1) = 0`, since given :math:`X=1`, :math:`Y` takes values 1 or 2
        each with probability :math:`\tfrac{1}{2}` — it cannot equal 0.
      - :math:`P(Y=0 \mid X=2)`: given :math:`X=2`, :math:`Y \sim \text{Poisson}(\lambda)`,
        so :math:`P(Y=0 \mid X=2) = e^{-\lambda}`.

      **Combining:**

      .. math::

         P(Y=0) &= 1 \cdot (1-p)^2 + 0 \cdot 2p(1-p) + e^{-\lambda} \cdot p^2 \\
                &= \boxed{(1-p)^2 + p^2\,e^{-\lambda}}.

      The answer is **(A)**.

.. admonition:: Problem 3 Setup
   :class: important

   Assume men's college basketball game lengths :math:`X` (minutes) are Normally distributed
   with mean 118 and standard deviation 10, where these values already account for fouls,
   timeouts, media breaks, and overtime.

   .. math::

      X \sim N(\mu = 118,\;\sigma = 10).

----

.. admonition:: Question 3a  (6 pts)
   :class: note

   Find the probability that a randomly selected game lasts within 1.5 standard deviations
   of the mean.

   .. dropdown:: Solution
      :class-container: sd-border-success

      :math:`X \sim N(\mu = 118,\;\sigma = 10)`. Standardize directly:

      .. math::

         P(\mu - 1.5\sigma < X < \mu + 1.5\sigma) = P(-1.5 < Z < 1.5).

      Using symmetry of the standard normal:

      .. math::

         P(-1.5 < Z < 1.5) = 2 \cdot P(Z < 1.5) - 1 = 2 \cdot 0.9332 - 1 = \boxed{0.8664}.

----

.. admonition:: Question 3b  (10 pts)
   :class: note

   Compute the interquartile range (IQR) for the population of men's college basketball
   game lengths.

   .. dropdown:: Solution
      :class-container: sd-border-success

      The IQR requires finding :math:`Q_1` (25th percentile) and :math:`Q_3` (75th percentile).

      **Step 1: Find the standard normal percentiles from the z-table.**

      For the standard normal, the 25th and 75th percentiles are symmetric about 0. From the
      z-table:

      .. math::

         z_{0.25} = -0.67, \qquad z_{0.75} = +0.67.

      **Step 2: Transform to the basketball game length distribution.**

      .. math::

         x_{0.25} = \mu + z_{0.25} \cdot \sigma = 118 + (-0.67)(10) = 118 - 6.7 = 111.3 \text{ min},

      .. math::

         x_{0.75} = \mu + z_{0.75} \cdot \sigma = 118 + (0.67)(10) = 118 + 6.7 = 124.7 \text{ min}.

      **Step 3: Compute the IQR.**

      .. math::

         IQR = x_{0.75} - x_{0.25} = 124.7 - 111.3 = \boxed{13.4 \text{ min}}.

.. admonition:: Question 3d  (3 pts)
   :class: note

   Consider three distribution models used in this course:
   :math:`\text{Normal}(\mu, \sigma^2)`, :math:`\text{Exponential}(\lambda)`, and
   :math:`\text{Uniform}(a, b)`. The interquartile range (:math:`IQR = Q_3 - Q_1`)
   measures the spread of the middle 50% of the distribution. In the statements below,
   "does not depend on the mean" means the IQR cannot be determined from the mean
   (:math:`E[X]`) alone; "constant multiple of the mean" means :math:`IQR = k \cdot E[X]`
   for some constant :math:`k` that does not vary. (If needed use scratch space on pg 2 of
   exam.)

   Which statement about the :math:`IQR` and the mean (:math:`E[X]`) is **incorrect**?

   - **(A)** For an Exponential distribution, the IQR is a constant multiple of the mean.
   - **(B)** For a Normal distribution, the IQR does not depend on the mean.
   - **(C)** For a Uniform distribution on :math:`[0, b]`, the IQR equals the mean.
   - **(D)** For a Uniform distribution on :math:`[a, b]`, the IQR is determined by the
     mean alone.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      Verify each statement:

      **(A) TRUE.** For :math:`\text{Exponential}(\lambda)`, the quartiles are obtained from
      the CDF :math:`F(t) = 1 - e^{-\lambda t}`:

      .. math::

         Q_1 = \frac{-\ln(0.75)}{\lambda} = \frac{\ln(4/3)}{\lambda}, \qquad
         Q_3 = \frac{-\ln(0.25)}{\lambda} = \frac{\ln 4}{\lambda}.

      .. math::

         IQR = Q_3 - Q_1 = \frac{\ln 4 - \ln(4/3)}{\lambda} = \frac{\ln 3}{\lambda}.

      Since :math:`E[X] = 1/\lambda`:

      .. math::

         IQR = \ln(3) \cdot E[X] \approx 1.0986 \cdot E[X].

      **TRUE** — IQR is a constant multiple (:math:`\ln 3`) of the mean.

      **(B) TRUE.** For :math:`\text{Normal}(\mu, \sigma^2)`:

      .. math::

         IQR = 2\,z_{0.75}\,\sigma = 2(0.67)\,\sigma = 1.34\,\sigma.

      This depends only on :math:`\sigma`, not on :math:`\mu = E[X]`. Knowing the mean tells
      you nothing about the IQR. **TRUE.**

      **(C) TRUE.** For :math:`\text{Uniform}(0, b)`:

      .. math::

         Q_1 = \frac{b}{4}, \quad Q_3 = \frac{3b}{4}, \quad IQR = \frac{b}{2}, \quad
         E[X] = \frac{b}{2}.

      Therefore :math:`IQR = E[X]`. **TRUE.**

      **(D) FALSE (the incorrect statement).** For a general :math:`\text{Uniform}(a, b)`:

      .. math::

         E[X] = \frac{a + b}{2}, \qquad IQR = \frac{b - a}{2}.

      These are two independent quantities. For example, :math:`\text{Uniform}(0, 4)` and
      :math:`\text{Uniform}(1, 3)` both have :math:`E[X] = 2`, but their IQRs are 2 and 1
      respectively. Knowing :math:`E[X]` alone does **not** determine the IQR for a general
      :math:`\text{Uniform}(a, b)`. The statement is **FALSE**.

Problem 4  (27 points)
------------------------------------------------------------------

.. admonition:: Problem 4 Setup
   :class: important

   Heekyung is training her cat, Meredith, to give a high-five. From her extensive
   experience, Meredith's response depends on whether a treat is offered. Heekyung offers a
   treat **70% of the time**. Meredith's behavior on an attempt is exactly one of the
   following: high-five, ignore, or nag.

   **If a treat is offered:**

   - Meredith gives a high-five with probability 0.8.
   - Meredith ignores Heekyung with probability 0.2.

   **If a treat is not offered:**

   - Meredith gives a high-five with probability 0.35.
   - Meredith ignores Heekyung with probability 0.6.
   - Otherwise, Meredith Nags Heekyung.

   Heekyung will restart the training next month if the probability that no treat was
   offered, given that a high-five occurred, is greater than one-half.

----

.. admonition:: Question 4a  (3 pts)
   :class: note

   Are the two events {Treat is offered} and {Give a high-five} independent? State yes or no
   and provide mathematical justification.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **No — the events are not independent.**

      .. math::

         P(\{\text{Give a high five}\} \mid \{\text{Treat is offered}\})
         \neq
         P(\{\text{Give a high five}\} \mid \{\text{Treat is not offered}\}),

      that is, :math:`0.8 \neq 0.35`, which in turn means the probability that Meredith gives
      a high five is dependent on whether Meredith is offered a treat or not.

----

.. admonition:: Question 4b  (2 pts)
   :class: note

   For each branch A, B, and C in the tree diagram below, write the probability statement
   and find its probability.

   .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/Exams/Exam1/FALL2025/image2.png
      :alt: A probability tree diagram. The root splits into Treat (probability 0.7) and Did not give a Treat (probability 0.3). From Treat, two branches lead to HighFive (probability 0.8) and branch A leading to Ignore. From Did not give a Treat, three branches lead to HighFive (probability 0.35), branch B leading to Ignore, and branch C leading to Nag.
      :align: center
      :width: 70%

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Branch A:**

      .. math::

         P(\{\text{Ignore}\} \mid \{\text{Treat is offered}\}) = 1 - 0.8 = \boxed{0.2}.

      **Branch B:**

      .. math::

         P(\{\text{Ignore}\} \mid \{\text{Treat is not offered}\}) = \boxed{0.6}.

      **Branch C:**

      .. math::

         P(\{\text{Nag}\} \mid \{\text{Treat is not offered}\}) = 1 - 0.35 - 0.6 = \boxed{0.05}.

----

.. admonition:: Question 4c  (4 pts)
   :class: note

   Determine the probability that Meredith Nags Heekyung.

   .. dropdown:: Solution
      :class-container: sd-border-success

      Apply the **Law of Total Probability**:

      .. math::

         P(\{\text{Nag}\})
         &= P(\{\text{Nag}\} \mid \{\text{Treat is offered}\})\cdot P(\{\text{Treat is offered}\}) \\
         &\quad + P(\{\text{Nag}\} \mid \{\text{Treat is not offered}\})\cdot P(\{\text{Treat is not offered}\}) \\[4pt]
         &= 0 \cdot 0.7 + 0.05 \cdot 0.3 = \boxed{0.015}.

----

.. admonition:: Question 4d  (8 pts)
   :class: note

   Determine the probability that Meredith gives a high-five.

   .. dropdown:: Solution
      :class-container: sd-border-success

      Apply the **Law of Total Probability**:

      .. math::

         P(\{\text{Give a high five}\})
         &= P(\{\text{Give a high five}\} \mid \{\text{Treat is offered}\})\cdot P(\{\text{Treat is offered}\}) \\
         &\quad + P(\{\text{Give a high five}\} \mid \{\text{Treat is not offered}\})\cdot P(\{\text{Treat is not offered}\}) \\[4pt]
         &= 0.8 \cdot 0.7 + 0.35 \cdot 0.3 = 0.56 + 0.105 = \boxed{0.665}.

----

.. admonition:: Question 4e  (10 pts)
   :class: note

   Based on your results, compute the probability that no treat was offered, given that a
   high-five occurred. According to these calculations and Heekyung's initial no retraining
   requirements stated at the start of this question, should Heekyung restart the training
   next month?

   .. dropdown:: Solution
      :class-container: sd-border-success

      Apply **Bayes' Rule**:

      .. math::

         P(\{\text{Treat is not offered}\} \mid \{\text{Give a high five}\})
         = \frac{P(\{\text{Give a high five}\} \mid \{\text{Treat is not offered}\})\cdot P(\{\text{Treat is not offered}\})}{P(\{\text{Give a high five}\})}.

      **Numerator:**

      .. math::

         0.35 \cdot 0.3 = 0.105.

      **Denominator** (from Question 4d):

      .. math::

         P(\{\text{Give a high five}\}) = 0.665.

      **Result:**

      .. math::

         P(\{\text{Treat is not offered}\} \mid \{\text{Give a high five}\})
         = \frac{0.105}{0.665} \approx \boxed{0.1579}.

      **Decision:** Do not restart training, as the probability is less than 0.5.

      .. admonition:: Grading note from instructors
         :class: danger

         The solution key notes: "Did not grade this part as we should have said will not
         restart training if probability is greater than 0.5." The condition as stated in the
         problem (restart if :math:`P > 0.5`) is confirmed as the intended interpretation.
         Since :math:`0.1579 < 0.5`, Heekyung does **not** restart training.

Problem 5  (28 points)
------------------------------------------------------------------

.. admonition:: Problem 5 Setup
   :class: important

   Purdue analyzed how long IT tickets take to resolve. Routine requests are handled in a
   quick "triage window" and are about equally likely to finish at any time during the first
   10 minutes. If a ticket survives past 10 minutes, the remaining time decays exponentially.
   Let :math:`T` be the resolution time (minutes) with PDF:

   .. math::

      f_T(t) = \begin{cases}
         k                 & 0 \leq t \leq 10 \\[4pt]
         k\,e^{-(t-10)}    & t > 10 \\[4pt]
         0                 & \text{otherwise}
      \end{cases}

   where :math:`k > 0` is to be determined.

----

.. admonition:: Question 5a  (10 pts)
   :class: note

   Determine :math:`k` so that :math:`f_T(t)` is a valid probability density function.

   .. dropdown:: Solution
      :class-container: sd-border-success

      :math:`k > 0`, otherwise :math:`f_T(t) < 0`.

      Solve for :math:`k` such that :math:`\displaystyle\int_{-\infty}^{\infty} f_T(t)\,dt = 1`:

      .. math::

         \int_0^{10} k\,dt + \int_{10}^{\infty} k\,e^{-(t-10)}\,dt = 1.

      .. math::

         10k + k = 1 \implies 11k = 1 \implies k = \boxed{\dfrac{1}{11}}.

----

The cumulative distribution function for the resolution time :math:`T` is given below,
defined up to the normalizing constant :math:`k` determined in part (a):

.. math::

   F_T(t) = \begin{cases}
      0                          & t < 0 \\[4pt]
      k \cdot t                  & 0 \leq t \leq 10 \\[4pt]
      1 - k\,e^{-(t-10)}         & t > 10
   \end{cases}

----

.. admonition:: Question 5b  (6 pts)
   :class: note

   What is the probability that a ticket is finished between 5 and 15 minutes? For your
   convenience, the CDF of the resolution time :math:`T` is given above, defined up to the
   normalizing constant :math:`k` determined in part (a).

   .. dropdown:: Solution
      :class-container: sd-border-success

      Substituting :math:`k = 1/11` into the given CDF:

      .. math::

         P(5 \leq T \leq 15) = F_T(15) - F_T(5).

      **F_T(5):** Since :math:`5 \in [0, 10]`, use :math:`F_T(t) = k \cdot t`:

      .. math::

         F_T(5) = k \cdot 5 = \frac{5}{11}.

      **F_T(15):** Since :math:`15 > 10`, use :math:`F_T(t) = 1 - k\,e^{-(t-10)}`:

      .. math::

         F_T(15) = 1 - k\,e^{-(15-10)} = 1 - \frac{e^{-5}}{11}.

      **Result:**

      .. math::

         P(5 \leq T \leq 15) = \left(1 - \frac{e^{-5}}{11}\right) - \frac{5}{11}
                              = 1 - \frac{5}{11} - \frac{e^{-5}}{11}
                              = \frac{6}{11} - \frac{e^{-5}}{11}
                              = \frac{6 - e^{-5}}{11}
                              \approx \frac{6 - 0.0067}{11}
                              = \frac{5.9933}{11}
                              = \boxed{0.5448}.

----

.. admonition:: Question 5c  (6 pts)
   :class: note

   Given that a ticket has been in the system for at least 5 minutes, determine the
   probability that the total time the ticket remains unresolved is at least 15 minutes.

   .. dropdown:: Solution
      :class-container: sd-border-success

      We want :math:`P(T \geq 15 \mid T \geq 5)`. Apply the definition of conditional
      probability:

      .. math::

         P(T \geq 15 \mid T \geq 5) = \frac{P(T \geq 15)}{P(T \geq 5)}.

      **P(T ≥ 5):**

      .. math::

         P(T \geq 5) = 1 - F_T(5) = 1 - k \cdot 5 = 1 - \frac{5}{11} = \frac{6}{11}.

      **P(T ≥ 15):**

      .. math::

         P(T \geq 15) = 1 - F_T(15) = 1 - \left(1 - k\,e^{-5}\right) = k\,e^{-5} = \frac{e^{-5}}{11}.

      **Result:**

      .. math::

         P(T \geq 15 \mid T \geq 5)
         = \frac{e^{-5}/11}{6/11}
         = \frac{e^{-5}}{6}
         \approx \frac{0.0067}{6}
         = \boxed{0.0011}.

      .. admonition:: Note
         :class: tip

         This is not the memoryless property (which would apply only in the purely
         exponential region). Here :math:`T \geq 5` includes the uniform region, so the
         conditional probability must be computed directly via the CDF.

----

.. admonition:: Question 5d  (6 pts)
   :class: note

   Only 5% of tickets take longer than :math:`t^*` to solve. Determine :math:`t^*`.

   .. dropdown:: Solution
      :class-container: sd-border-success

      Solve for :math:`t^*` such that :math:`P(T > t^*) = 0.05`.
      :math:`t^*` must fall into region :math:`t > 10` because
      :math:`F_T(10) = \dfrac{10}{11}`, which is less than 0.95.

      .. math::

         1 - F_T(t^*) = 0.05

      .. math::

         \frac{e^{-(t^*-10)}}{11} = 0.05

      .. math::

         -(t^* - 10) = \ln(0.55)

      .. math::

         t^* = 10 - \ln(0.55) \approx \boxed{10.5978 \text{ min}}.
