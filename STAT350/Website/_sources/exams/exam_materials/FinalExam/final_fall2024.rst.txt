Problem 1 — True/False (18 points, 2 pts each)
-----------------------------------------------

.. admonition:: Question 1.1  (2 pts)
   :class: note

   Given a dataset that contains multiple real outliers, the best measures of spread for this data would be the interquartile range (IQR).

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      When a dataset contains multiple real outliers, the IQR is the preferred measure of spread because it is resistant to extreme values. The IQR is based on the middle 50% of the data (Q3 − Q1) and therefore is not pulled toward outliers, unlike the standard deviation or range which are both heavily influenced by extreme observations.

.. admonition:: Question 1.2  (2 pts)
   :class: note

   Let :math:`X` follow a binomial distribution with fixed :math:`p` and sufficiently large number of trials :math:`n`. The estimator :math:`\hat{p} = X/n`, representing the sample proportion of successes, is derived from the random variable :math:`X`, which is the sum of :math:`n` independent and identically distributed Bernoulli trials. The sampling distribution of :math:`\hat{p} = X/n` is approximately normal.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      Since :math:`X \sim \text{Binomial}(n, p)`, we can write :math:`X = B_1 + B_2 + \cdots + B_n` where each :math:`B_i \sim \text{Bernoulli}(p)` is i.i.d. By the Central Limit Theorem, for sufficiently large :math:`n` (specifically, when :math:`np \geq 10` and :math:`n(1-p) \geq 10`), the standardized sum — and therefore :math:`\hat{p} = X/n` — is approximately normally distributed.

.. admonition:: Question 1.3  (2 pts)
   :class: note

   The time it takes for a customer to complete a transaction at a store follows an exponential distribution with a rate of :math:`\lambda = 1.4` transactions per minute. The probability that a transaction lasts less than 1 minute is greater than the probability that a transaction lasts between 1 and 3 minutes.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      With :math:`T \sim \text{Exponential}(\lambda = 1.4)`:

      .. math::

         P(T < 1) = 1 - e^{-1.4 \times 1} = 1 - e^{-1.4} \approx 0.7534

      .. math::

         P(1 < T < 3) = e^{-1.4 \times 1} - e^{-1.4 \times 3} = e^{-1.4} - e^{-4.2} \approx 0.2316

      Since :math:`0.7534 > 0.2316`, the statement is TRUE.

.. admonition:: Question 1.4  (2 pts)
   :class: note

   The time it takes for a customer to complete a transaction at a store is uniformly distributed between 2 and 10 minutes. The probability that a transaction lasts between 1 and 6 minutes is greater than the probability that a transaction lasts between 6 and 10 minutes.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      With :math:`T \sim \text{Uniform}(2, 10)`, the distribution has support only on :math:`[2, 10]`.

      .. math::

         P(1 < T < 6) = P(2 < T < 6) = \frac{6 - 2}{10 - 2} = \frac{4}{8} = 0.5

      .. math::

         P(6 < T < 10) = \frac{10 - 6}{10 - 2} = \frac{4}{8} = 0.5

      The two probabilities are equal (both 0.5), so the statement is FALSE.

.. admonition:: Question 1.5  (2 pts)
   :class: note

   A research team investigates whether consuming a spoonful of apple cider vinegar before meals prevents blood sugar spikes. They selected 30 pairs of identical twins, randomly assigning each twin in a pair to one of two groups. In this scenario, a two-sample independent procedure is appropriate to compare the groups.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      Because the study uses pairs of identical twins, the two observations within each pair are not independent — they share genetic and environmental factors. The appropriate procedure is a **paired (dependent) samples t-test** on the within-pair differences, not a two-sample independent procedure.

.. admonition:: Question 1.6  (2 pts)
   :class: note

   In a one-way ANOVA analysis for a factor with nine levels, the F-test resulted in rejection of the null hypothesis. If all possible pairs of levels are to be compared, the Multiple Comparisons step would involve 72 paired comparisons.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      The number of unique pairwise comparisons among :math:`k = 9` groups is:

      .. math::

         \binom{9}{2} = \frac{9!}{2! \cdot 7!} = 36

      There are 36 pairs, not 72. (The value 72 would arise from :math:`9 \times 8 = 72`, which counts each pair twice.)

.. admonition:: Question 1.7  (2 pts)
   :class: note

   The 98% lower confidence bound on the true slope of a simple linear regression line, :math:`\beta_1`, gives the value of −0.43. Then there is 0.98 probability that :math:`\beta_1` is greater than −0.43.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      Once the lower confidence bound has been computed from the data, it is a fixed number (−0.43). The true parameter :math:`\beta_1` is also a fixed (though unknown) constant — it is not a random variable. Therefore it is incorrect to assign a probability to the event ":math:`\beta_1 > -0.43`." The correct interpretation is: "We are 98% confident that :math:`\beta_1 > -0.43`," which is a statement about the long-run reliability of the procedure.

.. admonition:: Question 1.8  (2 pts)
   :class: note

   In simple linear regression, all influential points must be outliers.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: FALSE**

      An influential point is one whose removal would substantially change the fitted regression line. A point can be highly influential simply by being an extreme value in the :math:`x`-direction (high leverage) — even if its :math:`y`-value falls very close to the fitted line. Such a high-leverage point would not be an outlier in the residual sense. Therefore, influential points do not have to be outliers.

.. admonition:: Question 1.9  (2 pts)
   :class: note

   In simple linear regression, prediction intervals are wider than confidence intervals for the mean response at the same value of the predictor variable.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: TRUE**

      A prediction interval must account for two sources of uncertainty: (1) the uncertainty in estimating the mean response :math:`E[Y \mid x^*]`, and (2) the natural variability of an individual observation around that mean (captured by :math:`\sigma^2`). A confidence interval for the mean response accounts only for source (1). Because prediction intervals include the additional :math:`\hat{\sigma}^2` term, they are always wider than the corresponding confidence interval at the same :math:`x^*`.

.. admonition:: Question 2.1  (3 pts)
   :class: note

   Select the expression that does **NOT** correctly represent the probability of the colored area in the Venn diagram shown below.

   .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/Exams/FinalExam/FALL2024/image1.png
      :alt: Venn diagram with three sets A, B, and C. Sets A and B overlap; C is disjoint. The shaded region is the part of A that does not overlap B, plus all of C.
      :align: center
      :width: 65%

   - (A) :math:`P(A \cup B) - P(B) + P(C)`
   - (B) :math:`P(A) - P(A \cap B) + P(C)`
   - (C) :math:`P(A) - P(B) + P(C)`
   - (D) :math:`P(A \cup C) - P(A \cap B)`
   - (E) :math:`P(A \cup B \cup C) - P(B)`

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (C)**

      The shaded region is :math:`(A \setminus B) \cup C`, i.e., the part of :math:`A` that does not intersect :math:`B`, together with all of :math:`C`. Its probability is :math:`P(A) - P(A \cap B) + P(C)`.

      - **(A)** :math:`P(A \cup B) - P(B) + P(C) = P(A) - P(A \cap B) + P(C)` — **TRUE** (by the inclusion-exclusion identity :math:`P(A \cup B) = P(A) + P(B) - P(A \cap B)`, so :math:`P(A \cup B) - P(B) = P(A) - P(A \cap B)`).
      - **(B)** :math:`P(A) - P(A \cap B) + P(C)` — **TRUE** (directly matches the shaded region).
      - **(C)** :math:`P(A) - P(B) + P(C)` — **FALSE**. This subtracts all of :math:`P(B)` rather than only the overlap :math:`P(A \cap B)`. It would give the wrong value whenever :math:`B` extends outside :math:`A`.
      - **(D)** :math:`P(A \cup C) - P(A \cap B)` — **TRUE** (since :math:`A` and :math:`C` are disjoint, :math:`P(A \cup C) = P(A) + P(C)`).
      - **(E)** :math:`P(A \cup B \cup C) - P(B)` — **TRUE** (since :math:`B` is entirely inside :math:`A \cup B \cup C`, this correctly removes :math:`B`'s contribution while retaining :math:`A \setminus B` and :math:`C`).

      Therefore **(C)** is the expression that does **NOT** correctly represent the shaded probability.

.. admonition:: Question 2.2  (3 pts)
   :class: note

   Fréchet distribution is a heavily skewed, right-tailed continuous distribution that is used for modeling extreme events such as earthquake magnitudes, daily rainfall totals, and large insurance claims. Which of the following statements is TRUE about Fréchet distribution?

   - (A) The mean is the largest among the measures of central tendency, followed by the mode and the median.
   - (B) A small sample size is adequate to apply the central limit theorem to the distribution of the sample mean.
   - (C) For samples from this distribution, the median and variance are recommended measures of central tendency and spread, respectively.
   - (D) The interquartile range (IQR) is preferred for describing the spread of a population with a Fréchet distribution because it is less sensitive to outliers.
   - (E) None of the above statements are TRUE for the Fréchet distribution.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      - **(A) FALSE.** For a right-skewed distribution the correct ordering is mode < median < mean. Option (A) states the mean is the largest (correct), but then claims it is "followed by the mode and the median" — implying mean > mode > median. This reverses the relative positions of the mode and median; the correct descending order is mean > median > mode.
      - **(B) FALSE.** The Fréchet distribution is heavily skewed with very heavy tails. A large sample size is required before the CLT provides a good normal approximation to :math:`\bar{X}`.
      - **(C) FALSE.** The median is a good resistant measure of center, but the variance is not resistant to outliers. For a heavily skewed distribution the IQR is the preferred spread measure, not the variance.
      - **(D) TRUE.** Because the Fréchet distribution is right-skewed with potential extreme values, the IQR is preferred over the standard deviation or variance as a measure of spread; it is not influenced by extreme observations in either tail.
      - **(E) FALSE**, since (D) is true.

.. admonition:: Question 2.3  (3 pts)
   :class: note

   Suppose :math:`X` is a random variable with :math:`E[2^X] = 16`, :math:`\text{Var}(X) = 32`, and :math:`E[3X + 2] = 8`. Let a new random variable :math:`Y` be defined as :math:`Y = 2^X - \frac{1}{4}X^2`. What is :math:`E[Y]`?

   - (A) 0
   - (B) 7
   - (C) 8
   - (D) 36
   - (E) None of the above

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (B) 7**

      **Step 1: Find E[X] from** :math:`E[3X + 2] = 8`.

      .. math::

         3E[X] + 2 = 8 \implies E[X] = 2

      **Step 2: Find E[X²] using the variance formula.**

      .. math::

         \text{Var}(X) = E[X^2] - (E[X])^2 \implies 32 = E[X^2] - 4 \implies E[X^2] = 36

      **Step 3: Compute E[Y]** using linearity of expectation.

      .. math::

         E[Y] = E\!\left[2^X - \tfrac{1}{4}X^2\right] = E[2^X] - \tfrac{1}{4}E[X^2] = 16 - \tfrac{1}{4}(36) = 16 - 9 = \boxed{7}

.. admonition:: Question 2.4  (3 pts)
   :class: note

   An ANOVA F-test was performed on a dataset with two treatment levels (:math:`k = 2`), resulting in a test statistic of :math:`F_{\text{TS}} = 3.28`. If the same dataset is used for a hypothesis test on the difference of means of the two levels, then :math:`t_{\text{TS}}^2 = 3.28` only if:

   - (A) The null value, :math:`\Delta_0`, is 0.
   - (B) The hypothesis test is a two-tailed :math:`t`-test.
   - (C) The observations within and across the two levels are assumed to be independent.
   - (D) The population variances are assumed to be equal, and the pooled variance estimate is used to construct the test statistic.
   - (E) All of the above must hold simultaneously.
   - (F) We cannot be certain because the :math:`F_{\text{TS}}` and :math:`T_{\text{TS}}` are test statistics for two different hypothesis tests, each with distinct assumptions and interpretations.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (E)**

      The algebraic identity :math:`F_{\text{TS}} = t_{\text{TS}}^2` holds between the one-way ANOVA F-test (with :math:`k = 2`) and the two-sample :math:`t`-test **only when all four conditions are simultaneously satisfied**:

      - **(A)** The null value must be :math:`\Delta_0 = 0`, so the ANOVA null (:math:`\mu_1 = \mu_2`) aligns with the :math:`t`-test null (:math:`\mu_1 - \mu_2 = 0`).
      - **(B)** The :math:`t`-test must be two-tailed, because squaring the test statistic loses directional information and corresponds to the upper-tail F rejection region.
      - **(C)** Independence within and across groups must be assumed by both tests.
      - **(D)** The pooled variance estimate must be used in the :math:`t`-test; the ANOVA MSE is exactly the pooled variance estimate :math:`s_p^2`.

      If any condition fails — for example, if the :math:`t`-test uses Welch's (unpooled) variance or is one-sided — then :math:`t_{\text{TS}}^2 \neq F_{\text{TS}}`. Hence all conditions must hold simultaneously.

.. admonition:: Question 2.5  (3 pts)
   :class: note

   Which of the following statements is true for simple linear regression?

   - (A) All diagnostics plots rely on the residuals/errors.
   - (B) The assumption that the response is a simple random sample (SRS) for each fixed value of the explanatory variable is easy to verify.
   - (C) A scatter plot can be used to assess both linearity and normality.
   - (D) A scatter plot can be used to assess both homogeneity of variance and linearity.
   - (E) All of the above are true for simple linear regression.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      - **(A) FALSE.** The scatter plot of :math:`Y` vs. :math:`x` is a diagnostic plot that does **not** rely on residuals; it is used to assess linearity directly from raw data.
      - **(B) FALSE.** The independence/SRS assumption cannot be verified from the data alone; it depends on the study design and data collection procedure.
      - **(C) FALSE.** A scatter plot can be used to assess linearity, but **not** normality of errors. Normality is assessed using a histogram or QQ-plot of the residuals.
      - **(D) TRUE.** A scatter plot can reveal whether the spread of :math:`Y` values around the apparent trend changes as :math:`x` changes (homogeneity of variance) and whether the trend itself appears linear.

.. admonition:: Problem 3 Setup
   :class: important

   A dietitian is studying the effectiveness of a new dietary supplement on weight loss. To evaluate its impact, the dietitian measures the weight of **44 individuals** before and after a **4-week regimen** with the supplement.

   The difference in weight (:math:`d` = weight after − weight before) is calculated for each participant. The dietitian wants to establish whether the supplement results in a **decrease in weight**. The dietitian would like the test to have **at least 90% power** to detect an average weight decrease of 6 lbs, which is deemed an important reduction.

   The test is conducted at a significance level of :math:`\alpha = 0.05`, with hypotheses:

   .. math::

      H_0\colon \mu_d \geq 0 \qquad H_a\colon \mu_d < 0

   For this paired test, the standard deviation of the differences, :math:`\sigma_d`, is assumed to be **known** with a value of **12 lbs**.

Problem 3 — Power Analysis (16 points)
----------------------------------------

.. admonition:: Question 3a  (3 pts)
   :class: note

   Select the correct option for what the purple shaded region in the power graph actually represents.

   - (A) The intern is correct; it is the power of the test in detecting an alternative :math:`\mu_{d_a} = -6`.
   - (B) The intern is wrong; it is in fact the probability of Type I error.
   - (C) The intern is wrong; it is in fact the probability of Type II error.
   - (D) The intern is wrong, and it is none of the above options.

   .. figure:: https://yjjpfnblgtrogqvcjaon.supabase.co/storage/v1/object/public/stat-350-assets/images/Exams/FinalExam/FALL2024/image2.png
      :alt: Power analysis graph showing two overlapping bell curves. The left curve is centered at -6 (the alternative distribution) and the right curve is centered at 0 (the null distribution). A vertical dashed line marks d_cutoff. The purple shaded region is the area under the null (right) curve to the right of d_cutoff.
      :align: center
      :width: 80%

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      The graph shows two normal curves: one centered at :math:`\mu_d = -6` (the alternative distribution) and one centered at :math:`\mu_d = 0` (the null distribution). The vertical dashed line marks :math:`\bar{d}_{\text{cutoff}}`.

      The purple region is the area **to the right of** :math:`\bar{d}_{\text{cutoff}}` **under the null distribution** (centered at 0). Since the rejection region for this left-tailed test is :math:`\bar{D} < \bar{d}_{\text{cutoff}}`, the purple region is the **non-rejection region under the null distribution**, which has area :math:`1 - \alpha = 0.95`.

      - **Power** = area to the *left* of :math:`\bar{d}_{\text{cutoff}}` under the *alternative* distribution — this is **not** what is shaded (wrong distribution, wrong tail).
      - **Type I error** (:math:`\alpha`) = area to the *left* of :math:`\bar{d}_{\text{cutoff}}` under the *null* distribution — this is **not** what is shaded (wrong tail).
      - **Type II error** (:math:`\beta`) = area to the *right* of :math:`\bar{d}_{\text{cutoff}}` under the *alternative* distribution — this is **not** what is shaded (wrong distribution).

      The purple region corresponds to none of these, so the intern is wrong and the answer is **(D)**.

.. admonition:: Question 3b  (13 pts)
   :class: note

   Using the R output below, calculate the power of the test to detect an average weight decrease of 6 lbs. Determine whether the sample size is sufficient to meet the dietitian's requirement for at least 90% power. Provide a detailed explanation of your calculations, including all steps and reasoning, to receive full credit. This includes computing :math:`\bar{d}_{\text{cutoff}}` and writing out full probability statements.

   .. code-block:: r

      > qnorm(p = 0.05, lower.tail = FALSE)
      [1] 1.644854
      > qt(p = 0.05, df = 43, lower.tail = FALSE)
      [1] 1.681071
      > qnorm(p = 0.1, lower.tail = FALSE)
      [1] 1.281552
      > qt(p = 0.1, df = 43, lower.tail = FALSE)
      [1] 1.301552
      > pnorm(1.671771, lower.tail = TRUE)
      [1] 0.9527153
      > pnorm(1.671771, lower.tail = FALSE)
      [1] 0.04728474
      > pnorm(-2.975653, lower.tail = TRUE)
      [1] 0.001461827
      > pnorm(-2.975653, lower.tail = FALSE)
      [1] 0.9985382
      > pt(1.671771, df = 43, lower.tail = TRUE)
      [1] 0.9490839
      > pt(1.671771, df = 43, lower.tail = FALSE)
      [1] 0.05091613
      > pt(-2.975653, df = 43, lower.tail = TRUE)
      [1] 0.002391402
      > pt(-2.975653, df = 43, lower.tail = FALSE)
      [1] 0.9976086

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: Power = 0.9527; yes, the sample size is sufficient.**

      Since :math:`\sigma_d = 12` is known, we use the **z-test** framework.

      **Step 1 — Select the critical value.**

      For a left-tailed test at :math:`\alpha = 0.05` with known :math:`\sigma`:

      .. code-block:: r

         > qnorm(p = 0.05, lower.tail = FALSE)
         [1] 1.644854

      We use :math:`z_{0.05} = 1.644854`.

      **Step 2 — Compute :math:`\bar{d}_{\text{cutoff}}`.**

      The rejection region is :math:`\bar{D} < \bar{d}_{\text{cutoff}}`, where:

      .. math::

         P\!\left(\bar{D} < \bar{d}_{\text{cutoff}} \mid \mu_d = 0\right) = 0.05

      Solving:

      .. math::

         \bar{d}_{\text{cutoff}} = \mu_0 - z_{0.05} \cdot \frac{\sigma_d}{\sqrt{n}} = 0 - 1.644854 \times \frac{12}{\sqrt{44}} = -2.9757

      **Step 3 — Set up the power probability statement.**

      The power at :math:`\mu_{d_a} = -6` is:

      .. math::

         \text{Power} = P\!\left(\bar{D} < \bar{d}_{\text{cutoff}} \mid \mu_d = -6\right) = P\!\left(Z < \frac{\bar{d}_{\text{cutoff}} - (-6)}{\sigma_d/\sqrt{n}}\right)

      .. math::

         = P\!\left(Z < \frac{-2.9757 + 6}{12/\sqrt{44}}\right) = P(Z < 1.671771)

      **Step 4 — Read the power from the R output.**

      .. code-block:: r

         > pnorm(1.671771, lower.tail = TRUE)
         [1] 0.9527153

      .. math::

         \text{Power} = \boxed{0.9527}

      **Conclusion:** Since :math:`0.9527 > 0.90`, the sample size of :math:`n = 44` **is sufficient** to achieve the dietitian's requirement of at least 90% power.

.. admonition:: Problem 4 Setup
   :class: important

   Halin is a new student in STAT 350 and has never coded in R before. To improve her skills, she spends at most **40 minutes** each weekday doing R self-study. Her daily workflow is as follows:

   - The time it takes until she runs into an error, denoted by :math:`T`, follows an Exponential distribution with an **average time of 25 minutes**.
   - If :math:`T > 40`, she does not run into an error during her study session that day.
   - If :math:`T \leq 40`, she encounters an error and attempts to debug it:

     - Debugging **succeeds** with probability **0.7**, after which she feels happy and ends her study session.
     - Debugging **fails** with probability **0.3**, and she immediately stops and goes to **office hours** for help.

   - Each day's workflow is independent of other days.

Problem 4 — Exponential Distribution and Binomial (30 points)
--------------------------------------------------------------

.. admonition:: Question 4a  (8 pts)
   :class: note

   What is the probability that Halin will complete her study session on a given weekday without encountering an error?

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: 0.2019**

      Since :math:`T \sim \text{Exponential}\!\left(\lambda = \frac{1}{25}\right)` (rate = 1/25, mean = 25):

      .. math::

         P(T > 40) = e^{-\lambda \cdot 40} = e^{-40/25} = e^{-1.6} \approx \boxed{0.2019}

.. admonition:: Question 4b  (12 pts)
   :class: note

   On a given weekday, what is the probability that Halin does not need to go to office hours?

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: 0.7606 (exact: 0.760569)**

      Halin avoids office hours in two mutually exclusive scenarios: (1) she finishes without an error (:math:`T > 40`), or (2) she encounters an error but debugging succeeds (:math:`T \leq 40` and debug success). Because these two events are mutually exclusive, the addition rule applies; expanding the second event by the multiplication rule gives:

      .. math::

         P(\text{No Office Hours}) = P(T > 40) + P(\text{Debug Success} \mid T \leq 40) \cdot P(T \leq 40)

      .. math::

         = e^{-1.6} + 0.7 \times (1 - e^{-1.6})

      .. math::

         = 0.2019 + 0.7 \times 0.7981 = 0.2019 + 0.5587 = \boxed{0.7606}

      (Exact, without intermediate rounding: :math:`0.760569`.)

.. admonition:: Question 4c  (10 pts)
   :class: note

   Suppose Halin has continued her independent study for **20 weekdays**. Use the information from the previous question to determine the probability that Halin will visit office hours **exactly 5 times** over the 20 weekdays.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: 0.2011**

      Let :math:`N` = number of office hours visits over 20 independent weekdays. Since each day is independent with:

      .. math::

         p = P(\text{Office Hours}) = 1 - 0.760569 = 0.239431

      we have :math:`N \sim \text{Binomial}(n = 20,\ p = 0.239431)`.

      .. math::

         P(N = 5) = \binom{20}{5} \times (0.239431)^5 \times (0.760569)^{15} = \boxed{0.2011}

Problem 5 — One-Way ANOVA (42 points)
--------------------------------------

.. admonition:: Question 5a  (3 pts)
   :class: note

   Which of the following assumptions is **NOT** required to perform one-way ANOVA? Assume a factor has :math:`k` levels.

   - (A) Population variances are equal across the :math:`k` groups.
   - (B) An independent sample is randomly drawn from each of the :math:`k` groups.
   - (C) Observations within each group are independent of observations in other groups.
   - (D) The sample sizes from each of the :math:`k` groups are the same.
   - (E) The sample means are normally distributed for each of the :math:`k` groups.
   - (F) All of the above assumptions are required.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      One-way ANOVA requires: (A) equal variances (homogeneity of variance), (B) independent random samples from each group, (C) independence within and across groups, and (E) normality of observations within each group (which, by the CLT, implies the sample means are approximately normal). However, **equal sample sizes are not required**. ANOVA can be performed with unbalanced designs (different :math:`n_i`). Therefore **(D)** is the assumption that is NOT required.

.. admonition:: Question 5b  (2 pts)
   :class: note

   Determine whether the homogeneity of variance assumption is valid or invalid. Mathematically support your answer.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: The equal variance assumption holds (VALID).**

      The rule of thumb checks whether the ratio of the largest to smallest sample standard deviation is at most 2:

      .. math::

         \frac{s_{\max}}{s_{\min}} = \frac{\sqrt{147.7942}}{\sqrt{68.16}} = \frac{12.1572}{8.2561} \approx 1.473 \leq 2

      Since the ratio is less than 2, the homogeneity of variance assumption is satisfied.

.. admonition:: Question 5f  (3 pts)
   :class: note

   Based on your conclusion, determine whether you should proceed to conduct a Tukey HSD test.

   - (A) Conduct the Tukey HSD test because it can identify specific pairs of means that are significantly different when the ANOVA results show a significant difference.
   - (B) Do not conduct the Tukey HSD test because the ANOVA results indicate that the population means are not significantly different.
   - (C) There is insufficient information provided to decide whether a Tukey HSD test should be conducted.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (A)**

      Because the ANOVA F-test rejected :math:`H_0`, we know that **at least one pair** of group means differs, but we do not know **which pairs** differ. The Tukey HSD test is a post-hoc multiple comparisons procedure that controls the family-wise error rate while identifying which specific pairs are significantly different. We proceed with the Tukey HSD test.

.. admonition:: Question 5g  (3 pts)
   :class: note

   Regardless of your conclusion for part (f), the researchers decided to conduct a Tukey HSD with an overall significance level of 5%. Let :math:`\text{df}_E` denote the correct degrees of freedom for error. Choose the correct Tukey parameter.

   - (A) ``qtukey(0.95, nmeans = 3, df = dfe, lower.tail=TRUE) = 3.342793``
   - (B) ``qtukey(0.95/2, nmeans = 4, df = dfe, lower.tail=TRUE) = 1.925889``

   .. dropdown:: Solution
      :class-container: sd-border-success

      **Answer: (D)**

      The Tukey HSD procedure requires ``qtukey(1 - α, nmeans = k, df = df_E)``. Here:

      - :math:`\alpha = 0.05`, so we use ``1 - α = 0.95``.
      - :math:`k = 4` groups (``nmeans = 4``).
      - :math:`\text{df}_E = 176`.

      Options (A) and (C) use ``nmeans = 3`` (incorrect — there are 4 groups). Options (B) and (C) use ``0.95/2`` (incorrect — the Tukey quantile is not halved). Only **(D)** has both the correct confidence level (0.95) and the correct number of means (4).

   Using the summary information and the Tukey parameter above, construct a 95% confidence interval for the true difference in the average yearly amount spent by customers exposed to the AI-powered Ad Campaign and those exposed to the Direct Mail Ad Campaign.    .. dropdown:: Solution

.. admonition:: Question 6b  (15 pts)
   :class: note

   The following summary statistics were realized from the data above:

   .. math::

      n = 9, \quad \bar{x} = 32.600, \quad \bar{y} = 5.4711

   .. math::

      \sum_{i=1}^{9} x_i y_i = 1570.9020, \quad \sum_{i=1}^{9} x_i^2 = 9668.3960, \quad \sum_{i=1}^{9} y_i^2 = 281.7746

   (i) Determine the slope (:math:`b_1`) of the least-squares regression line.

   (ii) Determine the intercept (:math:`b_0`) of the least-squares regression line.

   (iii) Write out the equation of the regression line.

   .. dropdown:: Solution
      :class-container: sd-border-success

      **(i) Slope:**

      .. math::

         S_{xy} = \sum x_i y_i - n\bar{x}\bar{y} = 1570.9020 - 9 \times 32.600 \times 5.4711 = 1570.9020 - 1605.2238 = -34.3218

      .. math::

         S_{xx} = \sum x_i^2 - n\bar{x}^2 = 9668.3960 - 9 \times (32.600)^2 = 9668.3960 - 9564.84 = 103.556

      .. math::

         b_1 = \frac{S_{xy}}{S_{xx}} = \frac{-34.3218}{103.556} = \boxed{-0.3314}

      **(ii) Intercept:**

      .. math::

         b_0 = \bar{y} - b_1 \bar{x} = 5.4711 - (-0.3314335)(32.600) = 5.4711 + 10.8047 = \boxed{16.2758}

      **(iii) Regression equation:**

      .. math::

         \hat{y} = 16.2758 - 0.3314 \, x_{\text{Meal Cost}}

      **Interpretation of slope:** For each additional dollar in meal cost, the estimated tip amount **decreases** by approximately **$0.33**.

.. admonition:: Question 6c  (6 pts)
   :class: note

   List the assumptions of simple linear regression that can be evaluated using diagnostic plots. For each assumption, specify all the diagnostic plots that can be used to assess it. To receive full credit, you must include all relevant plots for each assumption.

   .. dropdown:: Solution
      :class-container: sd-border-success

      Three of the four LINE assumptions can be assessed graphically:

      **Linearity:** The relationship between :math:`x` and :math:`Y` is linear.

      - *Diagnostic plots:* **Scatter plot** (of :math:`Y` vs. :math:`x`) and **Residual plot** (of residuals :math:`e_i` vs. fitted values :math:`\hat{y}_i`). A non-linear pattern in either plot indicates a violation.

      **Homogeneity of variance (Equal variance):** The variance of the errors :math:`\epsilon_i` is constant for all values of :math:`x`.

      - *Diagnostic plots:* **Scatter plot** and **Residual plot**. Fanning or systematic changes in spread across the plot indicate heteroscedasticity.

      **Normality of errors:** The errors :math:`\epsilon_i` are normally distributed.

      - *Diagnostic plots:* **Histogram of residuals** (should be approximately bell-shaped) and **Normal QQ-plot of residuals** (points should follow the reference line).

      .. admonition:: Note
         :class: tip

         The **Independence** assumption (errors are independent) cannot be assessed from standard diagnostic plots; it depends on the study design and data collection.
